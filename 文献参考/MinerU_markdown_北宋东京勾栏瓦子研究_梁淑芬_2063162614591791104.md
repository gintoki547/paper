# 中文摘要

北宋时期是中国古代经济文化高度发达的时期，首都东京的文化产业出现了空前的繁荣。勾栏瓦子作为固定的娱乐演出场所，是市民阶层经常光顾之地。同时，勾栏瓦子也是一个专门的、综合性的大众休闲消费活动场所。

本文以北宋东京勾栏瓦子为研究对象，全文共分五个部分。首先对勾栏瓦子进行概括性总结得出自己的结论。主要从经济、政治、文化三个方面，研究分析北宋东京勾栏瓦子的兴起原因。勾栏瓦子是北宋东京著名的休闲娱乐场所，通过分析其空间分布、数量、演出活动分类以及经营管理方式等发展状况，以此来总结出勾栏瓦子在市民生活中的作用。从而进一步研究勾栏瓦子的发展状况。北宋东京勾栏瓦子的特征主要有大众娱乐性、分散性、商业性、专业性，正是这种特征吸引了广泛的社会群体在此消费娱乐，大大地改善了市民的生活质量。其消费群体涵盖了官僚及士大夫、驻军军士及其家属和普通大众等社会各个阶层。勾栏瓦子的主要社会功能集中表现在是市民休闲娱乐的场所，艺人的演出场所和商品交易场所，这些功能促进了东京城市功能和城市结构向休闲娱乐方面倾斜，从某种意义上来说，更加注重市民生活质量的提高和参与社会经济生活的能力。北宋东京勾栏瓦子是中国古代最早的休闲娱乐场所，对现代娱乐场所和娱乐活动的发展具有历史指导意义。如：勾栏瓦子逐渐演变为不同的休闲娱乐场所和演出场所、“说话”以及由此而产生的话本，推动了评书及小说的产生、杂剧促使了现代戏剧的诞生等，从而奠定了勾栏瓦子在中国社会文化史中的重要地位。

关键词：北宋；东京；勾栏；瓦子

# 前言

# （一）选题意义

北宋东京的社会经济和文化发展，在中国封建社会发展史上达到前所未有的高度。著名史学家陈寅恪先生曾说：“华夏民族之文化，历数千载之演进，造极于赵宋之世”， $^{①}$ 高度评价了北宋时期文化的发展在中国文化发展历程中所处的地位。清人严复先生认为：“古人好读前四史，亦以其文字耳。若研究人心政俗之变，则赵宋一代历史，最宜究心。中国所以成于今日现象者，为恶为善，姑不具论，而为宋人之所造就什八九，可断言也”， $^{②}$ 概括了宋代对中国今日之社会风俗、生活习惯、思想意识等的形成，起到了巨大的影响作用，甚至有学者认为北宋时期为中国的文艺复兴时期。

北宋时期社会经济发达，都城东京更是全国的政治、经济、文化中心，聚集了大量的物质财富和精神财富，更有一批德才兼备的杰出人才，使东京成为当时国际性的大都市，这些因素都孕育了勾栏瓦子的诞生。北宋东京勾栏瓦子是市民闲暇时间的娱戏之所，致使东京城市各种娱乐场所宾客盈门，娱乐活动丰富多彩。勾栏瓦子作为固定的娱乐表演场所，是北宋时期独有的文化现象，它在社会文化和市民生活方面所产生的影响是深远的。

研究勾栏瓦子，可以从侧面了解北宋时期的社会生活、市民生活、文化特征等，从而管窥北宋东京的城市发展全貌。同时，从社会学、文化学、休闲学的角度，来研究勾栏瓦子对当时乃至现代社会所产生的影响，比如：勾栏瓦子内各种活动的变迁，丰富了今日的休闲娱乐活动。勾栏瓦子的兴起带动了北宋东京休闲娱乐活动的飞速发展，丰富了市民的业余生活。勾栏瓦子的发展也促使了城市休闲娱乐功能的加强和城市空间结构的变化，这是同时期其它社会文化现象所不能比拟的。

研究北宋东京勾栏瓦子的发展状况，对于我们研究今天的娱乐活动和娱乐行业的发展仍有借鉴意义，如勾栏瓦子演出活动与今日娱乐活动之间的传承关系，以及它所带来的城市功能的转变和城市空间结构的变化对现代城市建设仍有一定的启示。因此，可以说，勾栏瓦子在我国社会文化史中具有不可替代的地位。

# （二）研究状况

目前，勾栏瓦子的研究在学术界日益受到重视，有关勾栏瓦子的研究主要表现在：①勾栏瓦子的概说以及内部活动的流变。②勾栏瓦子内演出活动和勾栏瓦子艺人发展状况。③勾栏瓦子的特征及其他。勾栏瓦子研究成果中相对较多的部分是勾栏瓦子内演出活动和瓦子艺人，关于勾栏瓦子这一现象的深入系统研究，以及勾栏瓦子在中国社会文化发展史中所处的地位，还很少有人注意到，以下从三个方面来叙述勾栏瓦子的研究状况：

# 1、勾栏瓦子概说性研究

周宝珠先生的《北宋东京研究》 $^{①}$ 一书，该书在第六章、第十二章、第十三章，分别研究勾栏瓦子的兴起，内部演出活动，但只注重宏观介绍，突出勾栏瓦子作为市民娱乐活动的场所带来市民生活的改变。

廖奔的文章《瓦舍勾栏考》 $^{②}$ 一文，该文较为全面的介绍勾栏瓦子的发展状况，全文共分八个部分，分别论述勾栏瓦子的称谓、历时、规模、地域、构造、表演内容、安排、经营管理，但文章只停留在宽泛的叙述和宏观介绍，对勾栏瓦子的特征、社会功能以及城市建设方面没有涉及到。康保成先生的《“瓦舍”、“勾栏”新解》 $^{①}$ 一文，侧重于探究瓦舍的来源，它从佛教的角度，对勾栏瓦舍“称谓”的来源进行了研究，得出了结论：勾栏瓦子最初是从佛教词汇和佛教用语里面发展而来的，也证明了勾栏瓦子是从唐代的寺院戏场发展而来。东方晓《南宋临安的瓦市》 $^{②}$ 一文，简单介绍南宋时期何谓“瓦市”、兴起的原因、数量、内部的演员及演出团体，并用较大篇幅介绍瓦子内的各种活动，对瓦子的经营和对社会影响方面的研究较少。人民教育出版社编审马执斌的《从乐府、教坊到瓦肆》 $^{③}$ 一文，主要研究北宋勾栏瓦子的发展渊源，即萌芽于晚唐，兴盛在两宋时期，其中还论证了乐府、教坊、瓦子之间的相互关系。

吴晟在勾栏瓦子方面的研究较为深入，其主要著作《瓦舍文化与宋元戏剧》， $^{④}$ 该著作前半部分详细介绍了勾栏瓦子的创立、繁荣、衰微时间与原因、瓦子内活动等。后半部分，使用较多的篇幅研究勾栏瓦子内各种伎艺活动的融合，带来了宋元戏剧的繁荣和发展，对宋元戏剧乃至现代戏剧的产生都起到了奠基性的作用。

吴晟另有三篇文章研究勾栏瓦子，其中《宋代瓦舍的创设及其文化意义》 $^{⑤}$ 一文，主要研究勾栏瓦子的创设历程，以及由此所产生的文化影响。另外两篇文章《瓦舍的衰微及其主要因素》 $^{⑥}$ 和《简析宋元民间戏剧观众的主要构成》 $^{⑦}$ ，前者主要论述勾栏瓦子衰微的原因。后者主要研究宋元戏剧的观众群体为军士、官吏、商人三种，他们不同的欣赏标准，促使了勾栏瓦子内演出活动的不断竞争和变化。吴晟对勾栏瓦子的研究主要在勾栏瓦子内演出活动与元杂剧的关系、特征、兴起与衰微等，侧重于勾栏瓦子内演出活动与元杂剧的关系，而对勾栏瓦子对市民生活所产生的影响，以及主要社会功能和城市功能建设方面的研究涉及较少。

陈雪飞《民间舞蹈商业化“剧场”雏形——宋代“勾栏”探析》 $^{①}$ 一文，主要研究勾栏瓦子为最初的“剧场”雏形，它所具有的具体特征和历史价值，其重点还是在于“剧场”单方面的研究。杜桂萍《论元杂剧与勾栏文化》 $^{②}$ 一文，认为元杂剧的发展是以勾栏为物质文化依托，为杂剧的发展提供了艺术底蕴，而大批文人的介入则促成了杂剧的繁荣和辉煌，但这还是侧重瓦子杂剧与戏剧的传承。湛江师范学院梁俊雄、徐剑峰、梁小珍合写的《宋明‘瓦舍’与现代‘健身娱乐场所’社会功能探析》 $^{③}$ 一文，主要研究勾栏瓦子与现代健身娱乐场所的传承关系，以及二者在功能上的一致性。

# 2、勾栏瓦子内演出活动的研究

在有关宋代文化、宋代市民生活、民风民俗以及北宋东京的研究著作里面，对勾栏瓦子内演出活动和艺人的发展状况都有提到。

董增刚的《市井瓦肆与生活》 $^{④}$ 一文，是一本较为全面的介绍瓦子内活动的著作，主要研究勾栏瓦子内部的演出活动，以及对市民生活的影响。伊永文的《行走在宋代的城市》 $^{⑤}$ 和尚园子、陈维礼合著的《宋元生活掠影》 $^{⑥}$ 两本书，提到了宋代市民生活视野下的勾栏瓦子，主要侧重于勾栏瓦子内演出活动的研究，对于它对市民生活与城市娱乐功能方面的影响较少提及到。石训、朱保书合著的《中国宋代文化》 $^{⑦}$ ，朱瑞熙、张邦炜、刘复生、蔡崇榜、王曾瑜合著的《辽宋夏金社会生活史》 $^{①}$ ，徐吉军、方建新、方健、吕凤棠合著的《中国风俗通史》 $^{②}$ ，这些著作只是对勾栏瓦子这一现象的描述和勾栏瓦子内各种演出活动的简单介绍，以上著作对勾栏瓦子本身的发展、特征、主要功能等方面的研究涉及较少。

上海师范大学胡语方的《瓦舍勾栏与市人小说》 $^{③}$ 一文，主要研究勾栏瓦子与市人小说的内容。文章前半部分研究勾栏瓦子在宋代的发展状况，包括数量、演出活动、瓦子艺人、以及各种表演活动的民间组织，后半部分研究勾栏瓦子的创设，为市人小说的发展提供了可能。舒焚的《北宋京瓦与京瓦艺人》 $^{④}$ 一文，主要研究北宋东京勾栏瓦子的发展，包括勾栏瓦子的性质、分布状况、演出活动、著名表演艺人、发展盛况以及最后被无情的战争结束使命。北京大学余江宁的文章《论宋代京城的娱乐生活与城市消费》 $^{⑤}$ 一文，主要介绍宋明勾栏瓦子的发展状况，以及城市消费水平及分类，也是很宽泛的研究。

以上研究主要集中在勾栏瓦子的演出活动和艺人的发展状况。

# 3、勾栏瓦子特征的研究

辽宁师范大学张楠的《从宋代“瓦肆”市场看我国古代商业音乐文化》 $^{⑥}$ 一文，主要研究勾栏瓦子内各种音乐节目，并注意到这些音乐节目所表现出的商业性、职业化、组织化、经营化的运作方式，折射出了两宋时期文化的繁荣状况和包容力，以及对市民阶层的社会影响。吴晟的《试论瓦舍文化的商业性与娱乐性》一文， $^{⑦}$ 主要研究勾栏瓦子的商业性功能和娱乐功能。郑州大学宋鸣笛《宋东京公共休闲空间研究》 $^{⑧}$ 一文，第四部分第三章节主要介绍宋东京勾栏瓦子的分布状况以及内部结构。该文将北宋东京城市内的所有公共休闲空间进行分类、列举、统计，这对本论文的写作有一定的借鉴作用。

# （三）研究方法

# 1、系统分析法

通过对勾栏瓦子的产生、发展、特征和主要社会功能的研究，进一步分析勾栏瓦子作为演出场所和休闲娱乐场所的主要作用及其流变等，本文拟在总结前人研究的基础上，在多角度、多层次的研究视野下，展现北宋东京勾栏瓦子的发展全貌。

# 一、北宋东京勾栏瓦子概说及其兴起原因

# （一）勾栏瓦子概说

北宋时期是科技、教育、文化、思想、艺术等飞速发展的时代，各个领域都取得了突破性的进展，具有典型意义的是四大发明有三项都是在宋代完成的。宋代也是市民阶层突飞猛进的发展时代，相应地，为市民阶层提供生活、生产的各种设施，就显得尤为重要。勾栏瓦子就是为了满足市民休闲娱乐需求而发展成为的一种大众休闲娱乐场所，为历代研究宋史的人们所关注。

瓦子，又称瓦肆、瓦舍、瓦市，萌芽于宋以前，兴起发展于北宋东京，南宋时期持续发展，直到元明清时期，逐渐走向衰落。勾栏瓦子的概念，历来评说不一，宋人吴自牧言：“瓦舍者，谓其‘来时瓦合，去时瓦解’之义，易聚易散也，不知起于何时。顷者京师甚为士庶放荡不羁之所，亦为子弟流连破坏之门”。 $^{①}$ 他认为：瓦子乃是来去自由，是京城士庶阶层娱乐、放松的地方，也是封建京城社会生活发展的产物。另一宋人耐得翁在《都城记胜·瓦舍众伎》条云：“瓦者，野合易散之意也，不知起于何时；但在京师时，甚为士庶放荡不羁之所，亦为子弟流连破坏之地”。 $^{②}$ 他认为：瓦舍乃是易聚易散，为士庶子弟流连破坏之地。二人的共同点在于：瓦舍不知起于何时，易聚易散、来去自由，在京师时，为士庶放荡不羁之所和弟子流连破坏之地。

现代辞书字典也对瓦子的概念做了多种诠释。主要有：民国74年，三民书局出版的《大辞典》解释“瓦子”为“宋代称妓院、茶楼、酒肆、卖杂货的店铺等场所”，这里从买卖、消费、经济方面来解释瓦子。中国文化研究所编印，中文大辞典编纂委员会编纂《中文大辞典》中解释“瓦舍”是“宋时妓院之称”（注：但据资料记载北宋时期是欠妥当的。关于“妓院”的称呼，起源于明代，明代统治者，将大量的政敌、部下及罪人的妻女收没其中，供军士消遣，勾栏瓦子即误认为是妓院）。北京商务印书馆1979年版《辞源》“瓦子”条说：“即妓院、茶楼、酒肆、娱乐、出售杂货等场所”。1986年汉语大词典编纂委员会编纂的《汉语大词典》中“瓦市”为：“宋、元、明都市中娱乐和买卖杂货的集中场所”。冯天瑜编《中华文化大辞典》中解释瓦子：“宋、元大城市里的娱乐场所”。综合以上释义可知，辞书字典中对瓦子的解释多为娱乐场所和集贸市场。

史学研究者对瓦子也有不同的看法。吴晟认为，“瓦”可能来自于古代流行于民间的一种乐器叫“瓦缶”，表演节目时的伴奏乐器，并一直流传，为“瓦肆”演变为娱乐场所提供了艺术根据。康保成先生考证，瓦舍原指寺院、僧房，勾栏又名“构栏”、“栏杆”，因在唐代时，寺院经常演出百戏伎艺，如唐长安的慈恩寺、安国寺、青龙寺等，都是最早的戏场，演出节目多为佛经有关的节目。周宝珠先生在《宋代东京研究》一书中写到：“瓦子是城市中大型的综合性的文艺演出场地”， $^{①}$ 即瓦子具有表演、娱乐、休闲放松功能的娱乐表演场所。周贻白先生认为：瓦舍“实则所指为旷场或原有瓦舍而被夷为平地，故名瓦子，也就是瓦砾场的意思”。 $^{②}$ 谢涌濠说：“瓦舍，是简易瓦房的意思，其含义指百戏、杂陈、百行云集的娱乐兼商贸市场”。 $^{③}$ 邓绍基则认为“也可以把‘瓦舍’之‘瓦’释为形状似瓦，即四周皆方，中间隆起”。 $^{④}$ 以上是现代学者对瓦子概念的各种解释，主要倾向于瓦子最初为类似瓦房的演出场所，至北宋时期发展为以娱乐为

主的集贸市场。

综合以上几种说法，可以得出结论：瓦子早期是一个简易的瓦房，供民间艺人在里面表演节目。经过历代发展，北宋时期发展成为一种集歌舞表演、弹唱、说话、商业贸易、集市等为一体的，具有商业性、娱乐性、大众性特征，为客人提供全方位服务的公共休闲娱乐场所。宋代以后，其功能不断分化并向纵身发展，发展成为不同的娱乐场所，如商业贸易发展为专门的集市；戏剧表演变为农村的戏台、戏楼和城市剧场；还有的演变为体育健身场所；消极的娱乐活动流变为妓院等。

勾栏又叫勾肆、构栏、游棚、腰棚、乐棚，原意为栏栅或栏杆之意，是瓦子里面不同的表演场地，也是瓦子的基本组成单元。最初的勾栏与表演没有任何关系，但这隔离开来的栏杆或勾栏，不允许人们随便进入，这就为它在宋代演变为演出场所，造就了有利条件，因为这样可以为表演者创造收入。勾栏之前的演出场所为露台，据《汉书》记载，露台始建于汉文帝时期，《汉书、卷四》记载内设戏台、戏房（后房）、腰棚（观众席），四周以栏杆包围，类似北宋勾栏，但此时只在宫廷演出。因此可知，露台是最早在宫廷的表演场所，露台演变为勾栏，并发展成为大众公共的休闲娱乐场所是在宋代才完成的。

勾栏瓦子具体兴起于何时，还没有确切资料记载，但据廖奔考证：“汴京的瓦舍勾栏兴起于北宋仁宗（1023-1063）中期到神宗（1068-1085）前期的十几年间”。 $^{①}$ 可以确定，勾栏瓦子兴起于北宋仁宗和神宗时期，地点就是当时的首都——东京汴梁。

# （二）勾栏瓦子兴起原因

# 1、城市格局改变，坊市界限打破

中国古代城市建设最初是为了其军事、政治目的，大多数城市都建城墙、开凿护城河等军事防御设施。其中，唐长安城是我国古代城市建设中最规整、最严密的建构格局典范，它在“原为隋代起建的新都大兴城” $^{①}$ 的基础上建造而成，城市建设之前制订详细的城市规划，修筑城墙、开辟街道、植树绿化等，都有严格的制度限制，整个长安市如同规整的棋盘见图一。城市建设实行严格的坊、市分离制度，即：居民区和集市区严格分离，各种商业活动只允许在固定的地方交易，市民则在特定的时间和地点，集中在一地进行商品交易。晚上则禁止任何商业活动，人们必须到自己所处的坊市内活动。

随着商品经济的发展，北宋东京市民临街设摊，走街串巷，早出晚归，活动于城市内的大街小巷，城市交通密集，道路两旁各种摊铺，依次排列，这一点我们可以从北宋著名画家张泽端的风俗画卷《清明上河图》中看到。然而，任何事情，都不会一帆风顺，临街设摊、占道经营也经历了不断的斗争。当时的占道经营的状况十分严重，宋仁宗天圣三年正月巡护惠民河官员田承说言：“河桥上多是开铺贩鬻，妨碍会及人马车乘往来，兼损坏桥道。望令禁止，违者重实其罪从之。……是月诏：“在京诸河桥上，不得令百姓搭盖铺占栏，有妨车马过往。” $^{④}$ 但往往是：禁令当天情况有所好转，几天之后一切如旧，街道上依然车水马龙。因此到宋仁宗景佑年间，允许市民临街开设店铺，从此延续了数百年的坊市制度彻底崩溃。

促使城市建构格局转换的另一重要原因是“鬼市”即夜市的出现。据记载：“乾德三年（965年）宣布不禁夜市，商业活动不再有时间限制开封城内十字大街有所谓‘鬼市’——五更点灯营业至天明；马行街北至新封丘门大街，夜市营业至三更，到五更时又再开张，至于‘耍闹去处’（娱乐场所如：勾栏、瓦子），营业‘通宵不绝’”。①自此人们的夜生活丰富起来，听戏、喝茶，酒楼、茶坊、瓦子、洗浴等等，改变了以往人们“日出而作，日落而息”的生活模式。

# 2、国家政局稳定，闲暇时间增多

北宋重文抑武的政策，自开国之日，就已经形成了，据记载宋太祖时期，皇宫有一处密室内立一石碑，《避暑漫抄》云：“太祖于太庙寝殿之夹室立一碑，谓之誓碑，……高七八尺阔四尺余，誓词三行，一云……。一云：不得杀士大夫及上书言事人。一云：子孙有逾此誓者，天必殛之”。 $^{②}$ 碑文曰：后世子孙不得杀士大夫及上书言事人，有违此誓者，天必诛之。又见史书记载：“上曰：王者虽以武功克定，终须用文德致治”。 $^{③}$ 可见，文治是宋初统治者的共识，于是“兴文教，抑武事”成为宋朝的一项基本国策。

由于北宋大力提倡以文德治理国家，使整个宋朝（北、南宋）时期是科举取士最多的朝代。据张希清的统计：“北宋贡举共开科考试81榜，其所取士正奏名进士19595人，诸科16366人。特奏名进士、诸科合计16035人。正奏名和特奏名进士诸科总计取士51996人。南宋贡举开科取士共49榜，正奏名进士23198人，特奏名进士19087人，共42285人。两宋通过科举总计取士115427人，为唐代的5倍，元代的30倍，明代的4倍，清代的3.4倍”。④两宋科举取士的总数115427人，是历代科举取士最多的朝代。同时，对科举人员的出身限制较小，太宗淳化三年朝廷下诏规定：“国家开贡举之门，广搜罗之路……如工商、杂类人内有奇才异行，卓然不群者，亦许解送”。 $^{①}$ 昭令显示：参加科举的人员，不仅包括官宦之家，而且工商、杂类人员才能出众者，均可以参加考试。

以文德治理国家和对文人的重用，促使北宋东京政局稳定，也促进了文化繁荣，东京汴梁曾出现文化夜市主要有：“酒楼茶坊的音乐演唱，瓦肆勾栏的杂剧艺伎表演，夜游天街观队舞”。②同时，也吸引了更多的人从事文化娱乐业，典型代表是图书出版业出现了前所未有的繁荣如：“综合而言，在图书事业方面，南方地区比北方地区发达，东部地区比西部地区发达。东京开封府为全国之最，西京河南府、南京应天府、京东路、京西路是北方发达之地，两浙、福建、江东、江西、成都府路是南方发达之地”。③

北宋东京，政治稳定，文化繁荣，使市民有了更多的闲暇时间，据《东京梦华录》记载东京的各种节日大致可以分为：时令节日、宗教节日、皇帝的寿诞等，各种节日加起来大致有100多个。北宋官员假期由祠部和鸿泸寺总领，大致分为公假和私假两大类。公假有节假、旬假、国忌假、外官上任假、唱名后假、朝假，还有特殊情况给假。私假有婚假、丧假、病假、探亲假、死忌假等。但是，由于宋代特殊的政治环境，各类节假日有所改变，但基本上都能满足官员的休假权利。如元丰五年（1082年），祠部复位休假制度，官员休假有：“元日、寒食、冬至，各七日；天庆节、上元节、天圣节、夏至、先天节、中元节、下元节、降圣节、腊日，各三日；立春、人日、中和节、春分、春社、清明、上巳、天祺节、立夏、端午、天贶节、初伏、中伏、立秋、七夕、末伏、秋社、秋分、授衣、重阳、立秋，各一日。另有大忌十五日、小忌四日。以上节假，加上每月有三日，全年三十六日的‘旬假’。加上大小忌和旬假共达124天”。 $^{①}$ 还有特殊情况，如旬假“建炎初，边事未宁，凡旬休及遇假曰：百司皆入局治事，其后，每旬唯以晦日休务”， $^{②}$ 即每逢10、20、30（29）日休假，其他时期官员可以“休务”。开宝九年（976年）宋太祖下诏：“己未，诏自今旬假不视事，赐百官休沐”。 $^{③}$ 旬假制度会因战事吃紧而取消，宋徽宗时期曾取消了一些假期。除了官员休假之外，对于在政府服役的丁夫和一般“工作人员”也给予一定的休假时间，如宋真宗景德四年，规定各官办的钱监铸匠每旬休假一天，如不愿休假，听其自便，此后大中祥符六年，又下诏“诸煎盐井役夫，遇天庆等四节并给假”。 $^{④}$ 宋宁宗时规定，凡“役丁夫”，元日、寒食、冬至、腊日各放假一日；宋宁宗时，“工作人员”每逢元日、寒食、冬至三大节各休假三日，逢圣节、每旬、请衣、请粮、请大礼赏各休假一日。 $^{⑤}$ 以上可知，宋代的休假制度使政府官员和一般官吏都拥有较多的闲暇时间。

除了以上假期之外，其它假期则更多如官员的婚假、丧假、病假，有的时间长达半年到三年之久。休假期间官员拥有较多的闲暇时间和经济实力，为勾栏瓦子的诞生提供了良好的社会环境。

# 3、市民阶层兴起，休闲空间不足

北宋东京市民阶层的兴起主要体现在：第一，被誉为中国民俗画的典范——《清明上河图》是一幅展现北宋东京市民在清明节时外出踏青的场景，画卷中描绘的人物有：商人、手工业者、摊贩、库利、脚夫、船夫、艺人、郎中、道士、尼姑、算命先生、乞丐、太学生、军士、外国使节等共684个，牲畜96头，房屋122座、舟船25艘等。第二，勾栏瓦子的消费群体构成也主要以市民阶层为多，大多数是品级低微的小官吏、农民、商人、僧人、道士、妇女、儿童、军士、文人士大夫（少数）等各色人等。

第三，宋人话本里的男女主人公，大多数也是普通市民、来往商贩和小手工业者。如：《错斩崔宁》中的崔小娘子是商人刘官人的小妾；《西山一窟鬼》的主人公吴教授是学堂里面的教书先生；《白娘子永镇雷峰塔》中的白娘子虽为蛇妖，却尽心帮助丈夫许宣打理生药铺；《志诚张主管》里的张胜是开线铺张员外家的主管；《金明池吴清逢爱爱》里的爱爱是开酒店人家的女儿；《碾玉观音》里的养娘秀秀家里是开裱褙铺的，而崔待诏(崔宁)则是个碾玉的人。这些主人公，虽说地位低下，生活贫苦，不受社会重视，但他们却是新兴的并逐渐扩大的普通市民阶层。他们思想进步、追求自由与幸福、和平与公正以及对美好生活和幸福爱情的向往，它反映了这一阶层市民的生活愿望，代表了一种社会发展方向，已经成为促进社会发展的主要力量之一。

经济发展，市民阶层兴起，人民生活水平提高，就注重精神享受。北宋的皇家园林、私家园林、寺庙、道观场所等，都不同时间的对外开放。如宋真宗咸平四年（1001年）“三月癸酉，枢密院言，准例春季金明池习水戏，开琼林苑，纵都人游赏，又大宴于含元殿”。 $^{①}$ 每年的三月一日至四月一日，皇家园林金明池向市民开放，金明池内彩楼林立、艺人勾肆、博易场户、关扑，出售各类衣袜鞋帽等用品不计其数，官民同乐呈现万民欢腾的局面。又如史书记载：“相国寺每月五次开放万姓交易，大三门上皆是飞禽猫犬之类，珍禽异兽，无所不有”。 $^{②}$ 相国寺每月五次向市民开放。

金明池和相国寺属于皇家园林，定期向市民开放，供人们在节日期间娱乐观赏。

虽然这些皇家园林或官家园林，在特定的时期供人们参观，但随着市民阶层兴起和闲暇时间的增多，现有的娱乐场所还是无法满足广大市民阶层的需求，勾栏瓦子便应运而生，满足了普通市民的娱乐需求。

# 4、商品经济发展，流动人口增加

北宋时期商品经济迅猛发展，物质财富急剧增加，造就了大量的流动人口，如商人、士大夫、手工业者、外来使节、僧人等城乡居民都涌入首都东京，为市井文化的繁荣打下了基础，也使北宋东京成为当时世界上人口最多的国家。这些人员在京师停留期间，工作之余需要消遣娱乐，寺院里面的戏场就无法满足市民需求，因此，固定的娱乐场所——勾栏瓦子便诞生了，成为真正意义上的商业娱乐表演场所。

北宋东京的商业发展在唐代时已经出现，据史书描述：“汴为雄郡，自江淮达于河洛，舟车辐辏，人庶浩繁”。 $^{①}$ 五代时期东京“控引河、汴，南通淮、泗，北接滑、魏，舟车之所凑集”， $^{②}$ 为宋东京的繁荣奠定了基础。

北宋东京，人口聚集，士庶浩瀚，物产之富庶，市井之繁华历代少见，见史书记载：

“都城之下，高门大第，宝货充积，富贵穷极，皆侍臣之所有也”。③

“东去乃潘楼街，街南曰‘鹰店’，只下贩鹰鹘客，余皆珍珠皮帛香药铺席。南通一巷，谓之‘界身’，并是金银彩帛交易之所，屋宇雄壮，门面广阔，望之森然，每一交易，动即千万，骇人闻见”。 $^{④}$ 

“咸平、景德以后，粉饰太平，服用寝侈，不惟士大夫家崇尚不已，市井

闾里以华靡相胜”。①

“国家承平岁久，兼并之民，徭役不及，坐取厚利，京城资产，百万者至多，十万而上，比比皆是”。 $^{②}$ 

“太平兴国六年（公元981年）汴河漕运江淮米三百万石，十多年后，至道初年增至五百八十万石，又十多年后，大中祥苻初年增至七百万石”。 $^{①}$ 

材料①②③和前页④描写京城市民资产雄厚，市井闾里以奢侈华靡相胜。材料④记载了官府也物产丰富，拥有雄厚的经济基础。由此可见，北宋东京城内下至市侩庶民，上至权公贵族皆资产雄厚，崇尚消费，消费品从一般生活用品转向奢侈品，珠宝古玩、珍奇之物，应有尽有。温饱之余，人民逐渐的注重享受，娱乐活动日益频繁。

随着经济的快速发展，随之衍生出许多相关的公门会所、交易交往模式和服务设施设备。政府为了促进商业的持续繁荣，专门成立了商税的管理部门——都商税院，设在义和坊，“掌京城商贾、廊店、市收”， $^{④}$ 专门维护商贾、店铺、小摊贩等的正常商业交易，并征收赋税。为了便于交易，各类“纸质货币”，如交钞、便换，以及会子、交子等货币交易媒介相继而生。各类茶馆、酒馆、驿站到处都是，满足了商人的日常生活所需。为了各地商人交流沟通方便，设立“会馆”，是每个地区的商人聚会议事的地方。所有这些因素都促进了商业的持续快速繁荣。难怪，很多历史学家，都将宋代称之为是“商业革命”的时代，如费正清和赖肖儿合著的《中国：传统与变革》一书中关于宋朝的介绍，其题目就是“商业革命”；斯塔夫里阿诺斯所著的《全球通史》中，对“宋朝的黄金时代”的标题下写道：“除了文化上的成就外，宋朝时期值得注意的是，发生了一场名副其实的商业革命，对整个欧亚大陆有重大意义”。由此可见，宋代时期发生了一场名副其实的商业革命。

经济的繁荣，促进了人口数量增加，但北宋东京人口到底有多少，目前还没有定论，多数学者认为有百万之多。周宝珠先生在《宋代东京研究》中提到：东京人口最盛时多达13.7万户左右，人口150-170万左右，主要包括驻军达20万之众（史书记载：“宋惩五代之弊，收天下甲兵数十万，悉萃京师” $^{①}$ ），皇室官府成员及为之服务的各类工匠约10万，还有未列入民籍的僧尼、道士近3万人，王公贵族蓄养的歌儿舞女不计其数，单是遍布全城的官妓私娼就在万人以上，使东京成为当时世界上人口最多的城市。且这些人员生活富裕、经济收入丰厚，有更多的闲暇时间和充足的经济购买力，导致了东京城内纯消费性人口的迅速增加。漆侠先生在《宋代经济史》中论证，北宋1350个有行政官署的城市中约有150座的人口超过万人，全国城市人口比重约占全国总人口的12%，而美国华裔学者赵冈则认为这一比率可高达20%。

# （一）勾栏瓦子的分布及其分类

# 1、北宋东京勾栏瓦子的分布

北宋东京城内瓦子分布广泛，宋人孟元老的《东京梦华录》中多处记载了北宋东京城市瓦子的分布状况，如：

“其御街东朱雀门外，西通新门瓦子以南杀猪巷”。 $^{①}$ 

“街南桑家瓦子，近北则中瓦，次里瓦。其中大小勾栏五十余座。内中瓦子莲花棚、牡丹棚，里瓦子夜叉棚、象棚最大，可容数千人”。 $^{②}$ 

“出旧曹门，朱家桥瓦子”。③

“街南蔡太师宅，西去州西瓦子，南自汴河岸，北抵梁门大街亚其里瓦，约一里有余”。 $^{④}$ 

“街西保康门瓦子，东去沿城皆客店，南方官员商贾兵级，皆于此安泊”。 $^{③}$ 

“马行北去旧封丘门外袄庙斜街州北瓦子，……处处拥门，各有茶坊酒肆，勾肆饮食”。 $^{⑥}$ 

“七月七夕，潘楼街东宋门外瓦子，州西梁门外瓦子”。⑦

材料显示，北宋东京共有瓦子10座，其中大型瓦子有6个分别是：

桑家瓦子、州西瓦子、州北瓦子、保康门瓦子、新门瓦子、朱家桥瓦子。其他小型瓦子有：桑家瓦子北的中瓦、里瓦、梁门大街外里瓦、宋门外瓦子，这类瓦子多依附于大型瓦子而存在。这些瓦子多集中在东京城的经济发达地带和中心娱乐区域，如御街、马行街、潘楼街、右掖门外街巷、大内前州桥东街巷等。除了以上瓦子之外，还有“路歧人”打野呵的地方，数不胜数，遍及东京城的各个角落。

# 2、勾栏瓦子的分类及内部结构

（1）瓦子名称辨析。瓦子又叫“瓦市”、“瓦肆”、“瓦舍”。

第一、瓦市。据康保成的《“勾栏”、“瓦舍”》新解，论证了瓦市是唐代的寺院戏场，唐代时期的寺院戏场史书记载：“长安戏场，多集于慈恩。小者在青龙，其次荐福、永寿。尼讲盛于保唐，名德聚之安国。士大夫之家入道，尽在咸宜”。 $^{①}$ 这里的慈恩、荐福、永寿、保唐、安国、咸宜均是寺院。寺院戏场到北宋时期发展为瓦市“东京相国寺乃瓦市也。僧房散处，而中庭两庑可容万人，凡商旅交易，皆萃其中，四方趋京师，以货物求售，转售他物者，必由于此”。 $^{②}$ 东京相国寺瓦市，每月开放五次，届时全国各地僧人、商人、文人、市侩等云集寺内，商品交易从珍禽鸟类、到生活日用品，一应俱全。扬宽先生也考证：“瓦子或瓦市，原是临时集市的意思。因为这种集市常以演戏的勾栏为中心，习惯上就把以勾栏为中心的集市称为瓦子或瓦市，如桑家瓦子就是以勾栏为中心的瓦市”。 $^{③}$ 由此可知：瓦市是一种以勾栏演出为中心的集市，除了表演之外，还兼具商业交易功能的贸易市场。

第二、瓦肆。北宋东京勾栏瓦子除了出售艺人的演出服务，如说话、绝活、杂技、唱赚等表演节目之外，另外还有摊贩、生意人在此进行商品交易。“瓦中多有货药、买卦、喝故衣、探搏、饮食、剃剪、纸画、令曲之类。终日居此，不觉抵暮”。 $^{④}$ 由此可以看出，瓦子不仅有艺人表演，还经营与此相关的商业活动。是一个以演出为主要目的，同时经营其他商业活动的场所，“瓦肆”由此而来。

第三、瓦舍，“舍”有住所的含义。勾栏瓦子艺人白天演出，晚上就有可能在瓦子内住宿，所以又通称“瓦舍”。材料记载：“监税以世禄为官，本亦粗识趋向，今其行为悖谬，赃污狼藉者，皆其妻有以致之也，阿除久居中瓦，耳濡目染，岂复有廉洁之行为……无非路岐杂剧人口中言语”。 $^{①}$ 文中描述的是监税官妻子阿除久居中瓦，以致于养成了勾栏瓦子艺人的生活习惯和行为方式，致使丈夫行为悖谬，以此可以推断勾栏瓦子艺人在瓦子内居住。另康保成《“勾栏”、“瓦舍”新解》一文，瓦舍原来是僧屋、寺院，因为其演出功能日益突出，演变为娱乐场所，而名称没有改变。由此可知瓦子也是供勾栏瓦子艺人休息居住的地方。

# (2) 勾栏的分类

瓦子内部用栏杆围起来供民间艺人演出的场地叫做 “勾栏”，它可以分为固定的和流动的两种。

固定的勾栏，一般设在瓦子内部，其表演时间、地点，甚至表演艺人、表演节目都是固定的，如小张四郎、史惠英等一世只在一个勾栏内演出，固定的勾栏属于主要的娱乐活动场所，但数量有限。

流动勾栏又叫做“勾肆”、“游棚”、“遥棚”、“棘盆”、“露台”、“乐棚”、“彩棚”、“彩楼”，分布在东京城的各个角落，满足了普通市民的娱乐需求。流动勾栏又可分为：节日里流动勾栏和平日里流动勾栏。节日里流动勾栏是指在节日期间，街道两旁或皇家园林内临时设置的勾栏、腰棚、露台、彩棚等，供人们在节日里欣赏娱乐，《东京梦华录》里记载了大量的节日里流行勾栏，如：

“殿上下回廊皆关扑钱物饮食伎艺人作场，勾肆罗列左右。……街东皆酒食店舍，博易场户，艺人勾肆，质库，不以几日解下，直至闭池，便典没出卖”。 $^{①}$ 

“自灯山至宣德门楼横大街，约百余丈，用棘刺围绕，谓之‘棘盆’，内设两长竿，高数十丈，以缯彩结束，纸糊百戏人物，悬于竿上，风动宛若飞仙。内设乐棚……。用黄罗设一彩棚……，面此乐棚。教坊钧容直、露台弟子、更互杂剧”。 $^{②}$ 

“门之两壁，皆高设彩棚，许士庶观赏，呈引百戏”。 $^{③}$ 

节日过后，这些流动性的勾栏就会自动拆除，街道恢复原来的面貌，继续从事商业或其它活动。

除了节日里流动勾栏，平日里的流动勾栏也遍布在城市各处，这类流动勾栏每天会在不同的地方演出，“诸门皆有宫中乐棚。……每一坊巷口，无乐棚去处，多设小影戏棚子”， $^{④}$ “如北瓦、羊棚楼等，谓之‘游棚’”。 $^{⑤}$ 平日里的流动勾栏，多分布在人流聚集处，如巷坊口、城门口、城郊、农村等地。这类流动性勾栏，表演节目简单、时间短，需要演员少，大多数是走街串巷的流浪艺人的表演场地。如史载：“王彭尝云：‘途巷中小儿薄劣，其家所厌苦，辄与钱，令聚坐听说古话’”。 $^{⑥}$ 材料描写的是只要观众围聚坐在一起，表演就可以开始，演出场地简单的令人吃惊，平日里流动的勾栏瓦子艺人被称为“路歧人”，他们的表演称为“打野呵”。

流动性的勾栏，丰富和补充了北宋东京的演出娱乐场所，是市民日常生活中较为常见的民间表演形式。

# (3) 瓦子与勾栏的内部结构

瓦子的内部结构比较简单，只要勾栏选好固定的位置，各勾栏依次排列分布在瓦子内部，大型的瓦子可容数十个勾栏，如“近北则中瓦，次里瓦。其中大小勾栏五十余座”， $^{①}$ “惟北瓦大，有勾栏一十三座”， $^{②}$ 由此可知：瓦子是由各个勾栏组成的。

勾栏内部结构比较复杂，其建设要考虑演出场地、演员用地、观众席、走道等位置。据吴晟研究：“在山西洪桐县赵城镇广胜寺明应王殿元剧壁画台口上方挂着一个帐额，其上写着‘尧都见爱大行散乐忠都秀在此作场泰定元年四月日’。可知，戏台正上方悬挂帐额，戏台是勾栏内高出地面演出的地方，戏台后半部分是“戏房”供演员化妆、更衣之用。勾栏一面有门，也叫“棚门”，供观众出入”。③

元杂剧《耍孩儿庄家不识勾栏》也记载：“要了二百钱放过咱，入得门上个木坡。见层层叠叠团峦坐座。抬头觑是个钟楼模样，往下觑却是个人旋窝。” $^{④}$ 描写了勾栏的内部情况：“钟楼模样”就是神楼，“人旋窝”则是腰棚。进人神楼或腰棚都通过楼梯，所谓“入得门上个木坡”，可知这些座位是阶梯状，但神楼和腰棚的价钱一样，由“要了二百钱放过咱”看出进门时付一次钱，进去后就可以随便坐。

另外，陈雪飞考证：“勾栏的基本形状是矩形或方形，四周有木板建成的墙体，屋顶为薄木板或草质材料做成；戏台靠勾栏剧场一端，为三面敞开的伸出式戏台，这一点和吴晟教授的研究相吻合；观众席是阶梯形的，并且围绕戏台呈三面环绕状，这也符合以自然声表演的最佳声学效果，使得所有观众都能接收到直达声，同时也符合文献所记载‘层层叠叠团团坐’的情形。勾栏瓦肆的观众区总面积在 450-600 平方米之间，加上戏台和周边的过道面积约 100 平方米，勾栏内部的面积约 625 平方米左右，近似长宽各为 25 米的正方形，或 20 米宽 30 多米长的矩形，戏台高度一般为 3-4 米，这和同时期神庙戏台的高度是一致的。勾栏的总高度应为戏台高度的两倍左右”。①陈雪飞研究得出结论：勾栏的形状为矩形或长方形，内有戏台和观众席，四周有墙体或者没有，屋顶多用木板或茅草搭盖，面积大约有 600 平方米左右。由此可看出，与吴晟教授和《庄家不识勾栏》里的记载相符合，可参见图三：

# （二）勾栏瓦子内演出活动的分类

勾栏瓦子做为民间的娱乐表演场所，除了继承前代已有的表演节目之外，又开发新创娱乐节目，汇聚了当时各类民间表演节目，同时也聚集了北宋东京著名的表演艺人。《东京梦华录》卷八《六月六日崔府君生日二十四日神保观神生日》条记载：“六月六日，……。自早呈拽百戏，如上竿、趋弄、跳索、相扑、鼓板、小唱、斗鸡、说诨话、杂扮、商谜、合生、乔筋骨、乔相扑、浪子、杂剧、叫果子、学像生、倬刀、装鬼、砑鼓、牌棒、道术之类，色色有之”。 $^{①}$ 文中描写北宋东京的娱乐节目主要有：上竿、趋弄、跳索、相扑、鼓板、小唱、斗鸡、说诨话、杂扮、商谜、合生、乔筋骨、乔相扑、浪子、杂剧、叫果子等精彩纷呈，令人目不暇接。勾栏瓦子内演出活动的分类按照演出地点的流动性和灵活性，可分为：固定演出和流动演出。固定演出主要是在勾栏瓦子内部表演，其表演时间和演员都是固定的。流动演出，主要是指节日期间，勾栏瓦子艺人在街道两旁或官私园林内临时搭建的彩楼、腰棚等演出场所。平日里还有一些伎艺稍逊的勾栏瓦子艺人不入勾栏在宽阔之处表演。另外，按照演出活动内容和表演形式可分为以下几种：

# 1、以说唱为主的表演活动

# (1) 说话

说话是瓦子里面主要表演节目之一，大致可分为四类：“说话者谓之“舌辩”，虽有四家数，名有门庭。且小说名‘银子儿’，如烟粉、灵怪、传奇、公案朴刀杆棒发发踪参之事，……谈经者，谓演说佛书。说参请者，谓宾主参禅悟道等事。……又有说诨经者。……讲史书者，谓讲说《通鉴》、汉、唐、历代书史文传，兴废争战之事。……盖小说者，能讲一朝一代故事，顷刻间捏合，与起令随令相似，各占一事也”。①由材料可知：说话分为讲史，即专讲朝代兴废之事。说经者，即演说佛书佛经之事。说参请者，即主讲参禅悟道之事。小说，又名“银子儿”，主要以灵怪、传奇、公案小说等取材广泛，如反映美好的爱情、向往天下太平，以及商人、手工业者幻想一夜暴富等内容。可以说，说话的内容遍及历史故事、参禅悟道、灵怪传奇、公案小说、爱情故事等等，表达了人们的美好愿望和心灵寄托。

说话表演时，通常一个人表演，时有夹带表演者演唱，或有乐器伴奏。勾栏瓦子里面的说话艺人都是当时名角，他们固定在一家勾栏，终日做场。《东京梦华录》载：“孙宽、孙十五、曾无党、高恕、李孝祥，讲史。李慥、扬中立、张十一、徐明、赵世亨、贾九、小说。……外入孙三神鬼。霍四九，说三分。尹常卖，五代史”。 $^{②}$ 孙宽、孙十五、曾无党、高恕、李孝祥、杨中立、张十一、徐明、赵世亨、贾九、霍四九、尹常卖等，都是备受欢迎的说话人，他们幽默风趣、出口成章，为说话艺术的繁荣发展做出了极大贡献。

# (2) 说诨话

说诨话又称十七字诗、长短句，类似现代节目的三句半，也是勾栏瓦子较受欢迎的说唱节目。据王灼《碧鸡漫志》记载：“长短句中，做滑稽无赖语。起于至和（1054-1056年），嘉佑（1056-1063年）之前，犹未盛也”，又载：“熙丰元祐间，衮州张三人以诙谐独步京师”，他编的诨话：“其词虽俚，然多颖脱，含讥刺，所处皆畏其口，争以酒食钱帛遣之”。③材料描写说诨话兴起于至和、嘉祐年间，“张山人，说诨话”， $^{①}$ 张山人是北宋东京著名的说诨话艺人，他的演出风格新颖、脱俗，多为讽刺性语言，当时曾誉满京师。

这些新兴的艺术表演形式，演出人员少、所需道具少，开场收场都非常简单，因此颇受艺人和观众的喜欢，因而是勾栏瓦子里常见的表演节目之一。

# (3) 弹唱

弹唱顾名思义，就是以弹和唱两种方式表演的娱乐节目。勾栏瓦子里面的弹唱节目主要有：小唱、嘌唱、唱赚，表演形式有个人演唱和多人合唱，大部分有乐器伴奏，这种亦唱亦演的弹唱形式备受观众欢迎，曾经红遍京师，并且影响到都城以外的城市和南宋临安。

周宝珠先生考证，小唱是北宋词曲配以音乐进行演唱的表演节目，北宋东京勾栏瓦子里面的小唱名角有：“主张小唱：李师师、徐婆惜、封宜奴、孙三四等，诚其角者”， $^{②}$ 且个个声音甜美，红遍京城。《都城纪胜》“瓦舍众伎”条云：“唱叫小唱，谓执板唱慢曲、曲破，大率重起轻杀，故曰浅酌低唱，与四十大曲舞旋为一体，今瓦市中绝无”。 $^{③}$ 这种浅酌低唱的表演旋律，声音软美，是一种小家碧玉型的音乐表演形式，表演时歌唱者手执拍板击拍，有时候有一人吹箫、觱栗伴奏，倍受欢迎。小唱不仅在勾栏瓦子里面表演，而且在酒楼、茶肆、妓馆均为人们喜爱的节目。

嘌唱也是勾栏瓦子的常见表演节目，宋人程大昌《演繁露》卷九记载：“凡今世歌曲，比古郑卫，又为淫靡，近又即旧声而加泛滟者，名曰嘌唱”。 $^{④}$ 这里将嘌唱理解为淫靡之声，歪曲了其本质意义。《都城纪胜》“瓦舍众

伎”云：

“嘌唱，谓上鼓面唱令曲小词，驱驾虚声，纵弄宫调，与叫果子、唱耍曲儿为一体，本只街市，今宅院往往有之。叫声，自京师起撰，因市井诸色歌吟卖物之声，采和宫调而成也。若加以嘌唱为引子，次用四句就入者，谓之下影带。无影带者，名散叫。若不上鼓面，只敲盏者，谓之拍打”。 $^{①}$ 

由此可知，嘌唱是融合了宫廷曲调、诸色饮叫之声和词曲合而为一体的歌唱形式，且唱法不同，如以嘌唱为引子的叫做“散叫”，若不用鼓面，只敲盏，则叫做“拍打”。由此可知，嘌唱演出时，一般有鼓、盏等乐器伴奏，将各种宫调和市井饮叫之声融合为一体的民间歌曲艺术形式，兴盛于东京，之后传遍全国各地一直到南宋时期。北宋东京的嘌唱明星有：“嘌唱弟子：张七七、王京奴、左小四、安娘、毛团等”， $^{②}$ 他们曾为皇帝演出过，是名冠京师的嘌唱名家。

唱赚融合了多种说唱伎艺手法的歌唱形式。因《东京梦华录》里面“其余不可胜数”，将唱赚给省略掉了，无从考证京师的唱赚名家和唱赚的演出状况，可见，唱赚当时并不是很流行。至南宋时期唱赚日益发展，据《都城纪胜》“瓦舍众伎”云：

“唱赚在京师日，有缠令、缠达：有引子、尾声为‘缠令’；引子后只以两腔互迎，循环间用者，为‘缠达’。……中兴后……赚者，误赚之义也，令人正堪美听，不觉已至尾声，是不宜为片序也。今又有‘覆赚’，又且变花前月下之情及铁骑之类。凡赚最难，以其兼慢曲、曲破、大曲、嘌唱、耍令、番曲、叫声诸家腔谱也”。③

由资料可推断，北宋东京唱赚主要包括：开头、结尾的引子，称为缠令，中间歌唱词曲称为缠达，在京师时较为简单，至南宋时期，发展成为一种集慢曲、曲破、大曲、嘌唱、耍令、番唱、叫声等诸色伎艺为一体的、新颖的、综合性的说唱艺术。南宋时期唱赚得到了广泛发展，产生了如濮三郎、扇李二郎、郭四郎、孙端等22位唱赚演员。唱赚表演时需要演员紧密配合，一般是演唱者击板、另有一人击鼓、一人吹笛伴奏。因此，唱赚者需要一定的技术功底、音乐天赋和良好的合作能力。

# (4) 吟叫

吟叫俗称叫果子，发源于市井之间，由诸色生意人歌叫买果子之声演变而来。宋人高承《事物纪原》卷9记载：

“嘉祐末，仁宗上仙，……然四海方遏密，故市井初有叫果子之戏。其本盖自至和嘉祐之间叫‘紫苏丸’，洎乐工杜人经‘十叫子’始也。京师凡卖一物，必有声韵，其吟哦俱不同，故市人采其声调，间以词章，以为戏乐也。今盛行于世，又谓之吟叫也”。 $^{①}$ 

材料描写北宋东京的卖物吟叫声，最初是由卖紫苏丸和叫果子开始的，它不是简单的唱叫，而是“采其声调，间以词章”，经过乐工的艺术加工而成，是吟叫伎艺产生的基础。因当时京师生意人颇多，他们为了招徕生意，而故意突出其叫卖声，形成了这风格独特，令人回味的音乐形式，符合当时的经济发展状况，满足了人民的娱乐需求。《东京梦华录》里面将吟叫称为“叫果子”，文八娘则是叫果子明星。

瓦子艺术是一种大众艺术，为了迎合市民口味，除了以上主要的音乐节目之外，还有融入俚语的其他说唱表演节目。如“合笙”“江浙间路岐伶女，有慧黠知文墨，能于席上指物题咏，应命辄成者，谓之合生；其滑稽含玩讽者，谓之乔合生。盖京都遗风也”。②由此可知，合生最早是酒筵间考验妓女才情的一种戏耍，后来普演于瓦子中。合生表演时观众可以当场指物赋诗，有点临时发挥的特点，又称“唱题目”，吴八儿乃是合生表演名角。合生节目对表演者的文化水平和应变能力要求极高，内容为滑稽讽刺之味的俗称“乔合生”。勾栏瓦子内的说唱表演形式多种多样，其他的就不一一举例。

# 2、以叙述故事为主的表演活动

# (1) 诸宫调

诸宫调是在原宫调的基础上发展而来，经过北宋艺人孔三传的艺术加工而成，在京师广为流传，“泽州孔三传首创诸宫调，古传士大夫皆能诵之” $^{①}$ ，“孔三传、耍秀才，诸宫调”。 $^{②}$ 孔三传是诸宫调的首创者，与耍秀才一起曾红遍京师。入南宋后，由熊保保以及“女童”继承，经过张五牛的改造，不断发展变化。诸宫调的演出方式见记载：“诸宫调本京师孔三传编撰，传奇、灵怪、入曲、说唱”。 $^{③}$ 可知诸宫调表演时有说有唱，以唱为主，由多个角色组成演绎完整的故事，并配有不同的乐器伴奏，常见的有水盏、鼓、笛、拍板等，表演时需要演员鼎力配合，北宋时期诸宫调还处于发展、变化的时期。

# (2) 杂剧

杂剧是在唐参军戏的基础上发展而来，宋仁宗以后，成了勾栏瓦子里面常见的表演节目。北宋东京较为流行和受欢迎的杂剧节目是《目连救母》，“构肆乐人，自过七夕，便般‘目连救目’杂剧，直至十五日止，观者增倍”。④《目连救母》能够连演7天，且观众越来越多，可见其精彩程度。杂剧节目的繁荣兴盛，也成就了一批杂剧艺人，“教坊减罢并温习：

张翠盖、张成弟子、薛子大、薛子小、俏枝儿、杨总惜、周寿奴、称心等”。①张翠盖、张成弟子、薛子大、薛子小等8人是杂剧的主要表演者，丁仙现曾是宫廷的御用艺人。

# (3) 杂扮

杂扮，又名杂班、纽元子、拔和，是杂剧的散段，杂剧正剧节目表演开始之前，先有幽默滑稽之人表演的散段子，以搞笑、幽默吸引观众为主。“刘乔、河北子、帛遂、吴牛儿、达眼五、重明乔、骆驼儿、李敦等，杂扮”。 $^{②}$ 可知从事杂扮表演节目的艺人很多，杂扮表演与正剧演出没有多少联系，可以根据时间长短而取舍。

南宋以后，诸宫调、杂剧、杂扮发展更是突飞猛进，并逐步相互融合得到了成熟、系统的发展。

# 3、以杂技为主的表演活动

# (1) 杂技

杂技，是现代称呼，杂技表演惊险、刺激，危险性极高，北宋东京勾栏瓦子杂技表演称为“杂手艺”载：

“张臻妙、温奴哥、真个强、没勃脐、小掉刀、筋骨上索杂手伎。浑身眼、李宗正、张哥、球杖踢弄”。 $^{③}$ 

“杂手艺皆有巧名：踢瓶、弄椀、踢磬、弄花鼓槌、踢墨笔、弄球子、拶（za）筑球、弄斗、打硬、教虫蚁，及鱼弄熊、烧烟火、放爆仗、火戏儿、水戏儿、圣花、撮药、藏压、药发傀儡、壁上睡、小则剧术射穿、弩子打弹、攒壶瓶、手影戏、弄头钱、变线儿、写沙书、改字”。 $^{④}$ 

资料显示，杂手伎也是勾栏瓦子的主要表演节目，种类繁多，如踢碗、弄瓶、踢磬、弄花鼓槌、踢墨笔、弄球子、拶筑球等。从节目名称和表演形式，可知每种杂手伎的表演方法，如弄瓶，可能是以瓶子为道具表演各种动作花样。其他有：壁上睡、烧烟火、剧术射穿、变线、写沙书、没勃脐、浑身眼、筋骨上索等，类似绝活、魔术。杂手伎表演时要求演员有扎实的基本功和不断的刻苦训练，否则无法完成高难度的动作，还有其它杂手技表演，就不一一举例。北宋东京的杂手技表演者主要有：张臻妙、温奴哥、李宗正、张哥等。

# (2) 口技

口技是以声音模仿各种禽鸟、动物的叫声见长的表演，其声音如真，嘹亮动听，并能招来大批飞禽，在市民中间普遍流行，也是瓦子里面的表演节目之一。《东京梦华录》卷九《宰执亲王宗室百官入内上寿》条载：“集英殿山楼上教坊乐人效百禽鸣，内外肃然，止闻半空和鸣，若鸾凤翔集”。 $^{①}$ 教坊乐人的口技表演可以招徕鸾凤翔集，逼真生动可以假乱真。

# 4、以竞赛为主的表演活动

# (1) 相扑

相扑，最初是军士业余时间为了消遣而进行的军事技术性表演活动，在军营中较为流行。“军头司每旬休按阅内等子、相扑手、剑棒手格斗”， $^{②}$ 由于其观赏性、竞争性和娱乐性，具有极强的攻击力，能满足市民求刺激、求输赢的心理特点，因而很快成为勾栏瓦子中常见的演出活动。同书卷五：“杨望京，小儿相扑、杂剧、掉刀、蛮牌”。 $^{③}$ 杨望京是小儿相扑能手。相扑至南宋时期，参与人员越来越多，“瓦市相扑者，乃路岐人聚集一等伴侣，以图摽手之资。先以女飐数对打套子，令人观堵，然后以膂力者争交”。①瓦子相扑表演时先有女子相扑手对打数个回合，以招揽观众，然后正式比赛开始，女子参与相扑表演增强了人们对相扑表演的热忠诚度。

# (2) 商谜

商谜是带有有奖竞猜性质的节目，由表演者出示谜题，观众竞猜，答对者可获得由出题人提供的奖品，类似有奖竞猜活动。《东京梦华录》载：“毛详、霍伯丑，商谜”。②毛详、霍百丑是北宋东京著名的商谜表演艺人，在瓦子里面广受欢迎。

商谜的种类繁多，主要有：“商谜者，先用鼓儿贺之，然后聚人猜诗谜、字谜、戾谜、社谜，本是隐语。有道谜，来客念思司语讥谜，又名‘打谜’。走智，改物类以困猜者。正猜，来客索猜。下套，商者以物类相似者讥之，又名‘对智’。贴套，贴智思索。横下，许旁人猜。问因，商者喝问句头。调爽，假作难猜，以走其智”。 $^{③}$ 可知，猜谜语的种类，有诗谜、社谜、戾谜，即来者索谜语、或者是表演者出题目，由来者竞猜，猜者也可以寻求帮助等。表演之前，先有表演者敲锣打鼓、然后聚人猜谜语，是一种具有互动性、参与性的表演节目，更加吸引观众。

# 5、其他表演节目

# (1) 影戏

影戏是中国由来已久的民间艺术，大多数人认为起源于汉代汉武帝时期，宋人高承《事物纪原》记载：“故老相承，言影戏之源，出于汉武帝李夫人之亡……，由是世间有影戏，历代无所见。宋朝仁宗时，市人有能谈三国事者，或探其说，加缘饰，作影人，始为魏蜀吴三分战争之像”。④影戏起源于汉代汉武帝怀念李夫人，官员做影人以解思念之情一事，至北宋仁宗时期，经过民间艺人的加工改造，发展成为勾栏瓦子内的演出节目。《梦梁录》“百戏伎艺”条云：“更有弄影戏者，元汴京初以素纸雕簇，自后人巧工精，以羊皮雕形，用以彩色装饰，不致损坏”。①影戏一般用纸或者羊皮做成各种人物的形象，通过光和幕布遮影成像，表演者在幕后操作，又称“皮影戏”、“弄影戏”、“手影戏”，同时讲述完整的历史故事或寓言传说。《东京梦华录》里记载：“董十五、赵七、曹保义、朱婆儿、没困驼、风僧哥、俎六姐，影戏。丁仪、瘦吉等，弄乔影戏”。②赵七、曹保义、朱婆儿、风僧哥都是影戏演员。乔影戏，根据乔合生的解释，以讽刺滑稽搞笑为内容的称为“乔合生”，以此推测影戏节目内容以讽刺滑稽搞笑的谓之“乔影戏”。影戏除了在勾栏瓦子表演之外，节日时间，每个巷坊门口，也多设小影戏棚子，供观众闲暇娱乐。

# (2) 傀儡戏

傀儡戏，民间称为木偶戏，大概起源于汉代前后，经过历代的不断发展，北宋时期成为勾栏瓦子里面常见演出节目，也成就了一批傀儡戏艺人。“般杂剧：杖头傀儡任小三，每日五更头回小杂剧，差晚看不及矣。悬丝傀儡：张金线。李外宁，药发傀儡”。③“又有一小船，上结小彩楼，下有三小门，如傀儡棚，正对水中……出小木偶人……继有木偶筑球舞旋之类，亦各念致语，唱和，乐作而已，谓之‘水傀儡’”。④傀儡戏分为药发傀儡、悬丝傀儡、水傀儡、杖头傀儡四种。张金线、李外宁、任小三都是北宋东京傀儡戏艺人。傀儡戏是由木材或其它材料做成的木偶人或动物，表演者用丝线进行控制，使其做各种各样的动作表演，表演题材多为历史故事、神话传说、寓言故事等，并配有声音，诙谐幽默，滑稽搞笑，受人欢迎。

# (3) 舞蹈

舞蹈是现代文艺表演的说法，北宋时期，舞蹈称之为“舞判”、“舞旋”等。周宝珠先生研究，东京比较流行的舞蹈有扑旗子、舞蛮牌、抱罗、舞判、七圣刀、哑杂剧、舞鲍老等多种。按人数可分为：单人舞、多人舞、对舞、车船舞、狮子舞等等。

勾栏瓦子表演场地相对较小，只适合单人表演的舞蹈，“张真奴，舞旋”， $^{①}$ 舞旋可能是单人表演的独舞，张真奴是勾栏瓦子舞旋名角。其他大型的集体舞蹈表演节目，需要演员多，演出场地面积大，一般在宫廷大型庆祝活动的时候出现较多，如《东京梦华录》里记载皇帝在宝津楼观看诸军百戏时就有场面宏大的对舞表演。

“乐部复动蛮牌令，数内两人出阵对舞，如击刺之状，一人做奋击之势，一人做僵仆。出场凡五七对，或以抢对牌，剑对牌之类”。 $^{②}$ 

“第一盏御酒，……。百官酒，三台舞旋，多是雷中庆。其余乐人舞者，浑裹宽衫，唯中庆有官，故展裹。舞曲破擷前一遍。舞者入场，至歇拍，续一人入场，对舞数拍。前舞者退，独后舞者终其曲，谓之‘舞末’”。③

“又爆杖一声，有假面长髯，展裹绿袍靴筒，如钟馗像者，傍一人以小锣相招和舞步，谓之‘舞判’。 $^{④}$ 

由此可知，北宋东京舞蹈多在宫廷重大活动或节日庆典期间表演，且规模宏大，是宫廷艺人和瓦子艺人相互促进演出水平的好机会。

# (4) 驯化动物

驯化动物节目有弄虫蚁、熊翻跟斗、驴舞蹈等，《东京梦华录》卷五条云：“刘百禽，弄虫蚁”， $^{①}$ 《归田录》载：“相国寺前，熊翻筋斗，望春门外，驴舞百枝”， $^{②}$ 这些都是经过驯化的动物。

由于北宋时期娱乐活动的演出场地没有明确界限，故对勾栏瓦子内的表演节目无法有准确的统计，以上将勾栏瓦子里面最主要活动列举出来，主要目的是为了印证勾栏瓦子演出活动的丰富多彩，表演形式生动活泼，受人喜爱。

# （三）勾栏瓦子的经营管理

# 1、勾栏瓦子艺人的分类及管理

# (1) 勾栏瓦子艺人的分类

宋杂剧中记载：勾栏瓦子演员，称为露台弟子或者女伶。露台是汉代时期宫廷的演出场所，后来人们就用艺人表演地点来称呼这些演员。伶，是在汉代时期演员的通称，宋代亦有使用。宋代勾栏瓦子的艺人可分为：教坊艺人、专业的瓦子艺人，路岐人、“和雇”艺人。

# 第一、专业的瓦子艺人

专业的勾栏瓦子艺人，是勾栏瓦子的专业表演者，他们以此为职业，终日在勾栏瓦子内演出。宋人孟元老《东京梦华录》里面就记载了大量勾栏瓦子的艺人，如李师师、张七七、任小三、杨望京、孙十五、霍四究、尹常卖、刘百禽等73位勾栏瓦子艺人，他们都是各自领域内的佼佼者，并且经常在官家、富家内演出，在东京城名声鹊起可谓当时“名角”，在后世留下叫得响的名字。专业的瓦子艺人是勾栏瓦子内演出的主力演员，也正是他们为东京市民带来了精彩的表演，为丰富民间艺术的种类做出了贡献。

# 第二、教坊艺人

宋以前的艺人分类有严格的限制，可分为两大类：宫廷艺人和民间艺人。宫廷艺人隶属于宫中教坊部管辖，民间艺人则相对比较松散，活动于大街小巷，两种艺人是绝对不能混淆的，因而也就没有任何交流机会。到了北宋时期，经济文化的高度繁荣，教坊部的职能逐渐弱化，宫廷艺人和民间艺人的界限逐渐模糊。“高宗建炎初省教坊，绍兴十四复置”，“绍兴三十年，有诏教坊即日黜罢，各令自便”， $^{①}$ “今虽有教坊之名，隶属修内司教乐所。然遇大宴等，每差衙前乐权充之，不足，则又和雇市人。近年衙前乐已无教坊旧人，多是市井歧路之辈”。 $^{②}$ 教坊地位的下降，教坊艺人不得不重谋生路，勾栏瓦子是他们演出的首选场所，既可以继续表演又可以养家糊口，一举两得，是多数宫廷艺人的谋生手段。

# 第三、“路岐人”

“路岐人”是指那些伎艺稍逊的勾栏瓦子艺人，无法进入瓦子内表演，只能在热闹宽阔之处做场，他们的表演成为“打野呵”，是流动勾栏的主要演出者，是对勾栏瓦子专业艺人的有益补充。

# 第四、“和雇”艺人

随着教坊职能的弱化，教坊艺人到勾栏瓦子内演出。那么，宫廷在重大节日庆典活动需要大量的演员时，怎么办？届时，教坊管理人员通过“和雇”的方式招集大量瓦子艺人到宫廷演出，并付给相应的酬劳。《武林旧事》卷四记载统计“乾淳教坊乐部”的演员情况来看，“和雇”演员共有161位，比宫廷中的正式演员多的多。“和雇”，交易雇佣也，是指宫廷在雇佣市井伎艺人的时候，引进市场机制，用钱来雇佣艺人，而不是像过去对待有乐籍的艺人那样，无偿地占有他们的劳动。北宋时期，瓦子艺人和教坊乐人是可以相互交换的，有时候也可以同台演出，《东京梦华录》卷五就记载了以上场景：“不以风雨寒暑。诸棚看人，日日如是。教坊钧容直，每遇旬休按乐，亦许人观看。每遇内前宴一月，教坊内勾集弟子小儿，习队舞，作乐杂剧节次”。 $^{①}$ 一遇重要节日，如正月十五“元霄节”、六月六“崔府君生日”、七月十五“中元节”等传统节日，勾栏艺人与教坊钧容直乐人同台演出，形成了“乐人时引万姓山呼”的壮观局面。

# (2) 勾栏瓦子艺人的管理

民间艺人主要在勾栏瓦子内表演，宫廷艺人走出宫廷到瓦子内演出，促进了两种艺人的融合与交流，也造就了艺术的繁荣。文献记载，这些艺人不管是宫廷的还是民间的，都属于教坊部管辖，《东京梦华录》“京瓦技艺”条记载：“崇、观以来，在京瓦肆伎艺：张廷叟，孟子书”，②《挥麈后录》卷四考证，“‘孟子书在宫廷里担任教坊乐官’张廷叟，则人份不明，可能为管理人员”，③《咸淳临安志》卷19亦云：“扬和王……今其屋在城外者多隶殿前司，城中着隶修内司”。由此可知，南宋时期，瓦子由殿前司和修内司分别管辖。以上可以推测，北宋东京时期，勾栏瓦子隶属于这些部门管辖，而张廷叟、孟子书则是教坊管理人员，负责勾栏瓦子的日常管理活动，专门对这些瓦子艺人进行管理，主持日常事务。除了教坊部管辖勾栏瓦子艺人之外，当地官员也对勾栏瓦子的经营负责管理。如：《水浒传》第五十回《插翅虎枷打白秀英》中李小二说：“都头（雷横）出去了许多时，不知此处近日有个东京新来打楚的行院……。那妮子来参都头，却值公差出外在外”， $^{①}$ 李小二的话说明瓦子艺人到某地演出时，须向当地的官员（都头）参拜，方可演出。

# 2、演出活动的广告宣传

首先，冠以名人名号

勾栏瓦子艺人为了提高声誉，扩大影响，用当时的官衔、或用科举功名名称作为自己的艺名，还有炫耀自己曾蒙受皇恩，并赐以封号，以此炫耀自己表演水平高超。如：《武林旧事》“各色伎艺人”条所载小说伎艺人中，诸伎艺人名字后面都加有“御前”二字，如任辩御前、施圭御前、叶茂御前、方瑞御前等，“御前”即曾在皇宫演出，皇帝御赐的“封号”，艺人以此来炫耀演出水平高超，以提高知名度。同书“乾淳教坊乐部”记载，许多乐人还拥有职衔，如武功大夫、忠顺郎、武德郎、承节郎、节级等，虽属“杂流命官”，但显示出艺人演出水平一流。勾栏瓦子中的普通艺人则用科举功名名称作为自己的艺名，如许贡士、张解元、周八官人、陈进士、林宣教、李郎中、陆进士等，其目的也是提高自己知名度，同时也促进了演出质量和演员素质的提高，增加演出团队的总体实力，增强竞争力，这是勾栏瓦子艺人在表演节目时广泛采用的宣传办法。

其次，服装艳丽、统一

服装鲜艳、华丽，着装统一，给观众以视觉上的冲击，便于人们留下深刻印象，这也是一种广告宣传方式。是北宋东京勾栏瓦子艺人经常使用的宣传方法，直到现在还广为使用。《东京梦华录》卷九记载：

“……小儿各选年十二三者二百余人，列四行，每行对头一名，四人簇拥，并小隐士帽，着绯绿紫青生色花衫，上领四契义襕束带，各执花枝。排定，先有四人裹卷角幞头、紫杉者，擎一彩殿子，内金贴子牌，擂鼓而进，谓之‘队名牌’，上有一联，谓如‘九韶翔彩风，八佾舞青鸾’之句……。第七盏御酒慢曲子，宰臣酒皆慢曲子，百官酒三台舞讫，参军色作语，勾女童队入场。女童皆选两军妙龄容颜过人者四百余人，或戴花冠，或仙人髻鸦霞之服，或卷曲花角幞头，四契红黄生色销金锦绣之衣，结束不常，莫不一时新装，曲尽其妙。杖子头四人，皆裹曲角向后指天幞头，簪花、红黄宽袖衫，义襕，执银裹头杖子。皆都城角者，当时乃陈奴哥、俎姐哥、李伴奴、双奴、余不足数。亦每名四人簇拥，多作仙童丫髻，仙裳执花，舞步进前成列。或舞《采莲》，则殿前皆列莲花。槛曲亦进队名。参军色作语问队，杖子头者进口号，且舞且唱。乐部断宋《采莲》讫，曲终复群舞”。 $^{①}$ 

此文中提到勾栏瓦子艺人的服装有：青色花衫、卷角幞头、紫衫、义襕、小隐士帽，簪花，鲜花等服饰，还有各种各样的佩饰，皆是一时新装。勾栏瓦子演员皆妙龄少女，年轻美貌、服装华丽，表演起来翩翩起舞、玲珑之声响彻于耳，给观众造成一种眼前的幻觉，如同走进梦幻般的场景，容易给人留下深刻的印象。

再次，招帖、帐额宣传

招贴、帐额，是勾栏瓦子艺人专门制作的，写着演出节目名称、演员名字，能让观众观看演出之前就对表演节目有所了解，多悬挂在勾栏正面，又称为“招幌”。如《庄稼不识勾栏》：“正打街头过，见吊个花碌碌纸榜”，纸榜就是“招幌”，上面写着勾栏内表演内容、演员名字。又如：《水浒全传》第五十一回记雷横和帮闲李小二来到勾栏里，看见门首挂着许多‘金字帐额’把当天所要讲唱的话本名目，也就是内容题材，展示出来。上写：“今日秀英招牌上明写着这场话本，是一段风流蕴藉的格范，唤做《豫章城双渐赶苏卿》”。 $^{②}$ 帐额上面写出女演员白秀英的名字、演出内容。山西洪桐县赵城镇广胜寺明应王殿元剧壁画，戏台口上方挂着一个帐额，其上写着‘尧都见爱大行散乐忠都秀在此作场泰定元年四月日’，这是帐额留下的实物见证。

可见，招贴、帐额是瓦子艺人常用的宣传方式之一，随着社会的不断发展，招贴、帐额宣传方式，外观制作更加美观，所用材料质量更加上乘，使用范围也越来越广。

第四，表演节目中，炫耀技艺高超

勾栏瓦子艺人表演节目之中，也对自己演出活动进行宣传，一般以吹嘘自己的说唱技巧，或是透露演出内容，或是故弄玄虚以引人注意，一定程度上也起到了宣传作用。

勾栏瓦子艺人在表演节目之前，都先有“致语”即开场白，好的致语也可以宣传演出节目。如《事林广记》载遏云社赚词的致语云：“遇酒当歌酒满斟，一觞一咏乐天真。三杯五盏陶情性，对月临风自赏心。环列处，总佳宾，歌声嘹亮遏行云。春风满座知音者，一曲教君侧耳听”。 $^{①}$ 致语表达的场景是：一杯茶、一杯酒，几个朋友，在月亮下面听着美妙的歌声，畅谈人生，是何等的赏心悦目，令人心旷神怡，自然就吸引人前来观看。因此，从某种程度来看，也宣传了演员及其演出活动。

也有在表演节目过程中宣扬自己表演水平高超的，如：《醉翁谈录》甲集卷一云：

“讲论处不滞搭，不絮烦；敷演处有规模、有收拾。冷淡处提掇得有家数，热闹处敷演得越久长。曰得词，念得诗，说得话，使得砌。言无讹舛，遣高士善口赞扬；事有源流，使才人怡身神嗟讶”，……“破尽诗书泣鬼神，发扬义士显忠臣”，又“春浓花艳佳人胆，月黑风寒壮士心，讲论只凭三寸舌，称平天下

浅和深”。①

文中描述的说话人对话本的熟练程度, 表演起来疏密有度、能合能分、分得轻重缓急, 衔接之处把握的流利、顺畅, 且诙谐幽默, 引人发笑, 自然就能吸引更多的观众前来观看。

节目表演结束时，为了吸引观众下次再来，表演者故意在情节紧张高潮之处结束，用一些故弄玄虚的语言吊人胃口如：“欲知后事如何，请听下回分解”，从某种意义上来说这也是一种宣传方法。

# 第五，鼓声宣传

勾栏瓦子表演节目之前，先鸣锣敲鼓，引人注意。《庄家不识勾栏》：见几个妇女向台儿上坐，又不是迎神赛社，不住的擂鼓筛锣。《南村辍耕录》卷二十四“勾阑压”载：“有女官奴，习讴唱，每闻勾栏鸣鼓，则入”，②《水浒传》第一百十回《燕青秋林渡射 宋江东京城献俘》中记载：“来到瓦子前，听的勾栏内锣响，李逵定要入去，燕青只得和他挨在人丛里，听的上面说评话，正说《三国志》，说到关云长刮骨疗毒”。由此可知，勾栏艺人表演之前，鸣锣敲鼓引人注意，使其前来观看表演，也是主要的宣传方法之一。

以上材料的叙述, 可以推测北宋东京时勾栏瓦子艺人已懂得用各种宣传手法, 形成了瓦子艺人之间自由竞争的经营模式, 有利于勾栏瓦子艺人水平的提高和形成良好的商业环境, 这在当时是很先进的。

# 3、配套的设施设备

随着北宋东京勾栏瓦子的不断发展壮大，为了便于观众观看演出和演员表演，瓦子内部已经出现各类配套的设施设备，具备了娱乐演出场所的

基本条件，其主要的设施设备有：

观众席，分为神楼和腰棚。神楼是正对戏台的位置为最佳位置，腰棚是从神楼开始围绕戏台往后延伸的座位，整个观众席围绕戏台呈三面环绕之状，其中“青龙头”是座位中间最好的位置，一般为富家或官家占用。这种环形观众席排列方法，符合自然声学的传声效果。

舞台，又称戏台。正对观众席的正前方，为了观众都能看到表演，戏台一般要高出地面，类似现代舞台。戏台的后面有戏房，供演员更衣使用，类似更衣间。戏台上还设有乐床，即放置乐器的地方。所有这些都方便了演员演出。

工作人员，多是负责维护勾栏瓦子的正常运转，如《庄家不识勾栏》里面的门口“把门要了二百钱放过咱”，这里收钱的人，类似剧场售票员和检票员，负责观众进场。另《南村辍耕录》载：“亦有被压而幸免者，见衣朱紫人指示其出，不得出者，亦曲为遮护云”。 $^{①}$ 描述的是一次勾栏倒塌事件中，有些没有受伤的观众因“衣朱紫人”的引导，而脱离险境，由此可见“衣朱紫人”是在紧急情况下疏散人群的工作人员，类似保安。

其他相关的设施设备，瓦子内部环形的观众席排列办法，符合自然学的传声效果。勾栏瓦子为了便于观众观看演出，还设立了其他一系列的辅助设施有：茶坊、起店、酒楼、店铺等，以供演出人员和观众在瓦子内饮食所需。另外还有出售演出内容的文字资料，如刊印小说，出售纸画、令曲（曲谱唱词的稿子等），其他剃剪、喝故衣、货药之类的摊贩和店铺也很多，极大的方便了观众在勾栏瓦子内的活动。

# 三、北宋东京勾栏瓦子的特征及其消费群体分析

# （一）勾栏瓦子的特征

# 1、大众娱乐性

勾栏瓦子的大众性表现在，第一演员种类繁多，可分为瓦子艺人、“和雇”艺人、路岐人等。第二，演出活动的平民化、世俗化。勾栏瓦子的表演节目，如说话、杂剧、唱赚、杂技、傀儡戏、影戏等，都通俗易懂，适合大众口味。第三，勾栏瓦子消费群体遍及社会的各个阶层，上至官僚、士大夫，下至平民百姓，他们在闲暇之余都可能在勾栏瓦子内消遣。

勾栏瓦子作为大众性的休闲娱乐场所，目的就是市民闲暇娱乐之地，具有极强的娱乐性如：

“小说起宋仁宗，盖时太平盛久，国家闲暇，日欲进一奇怪之事以娱之”。 $^{①}$ 

“顷在汴京时，村落野夫，罕得入城，遂撰此端。多是借装为山东、河北村叟，以资笑端”。 $^{②}$ 

“闻刘玄德败，颦蹙有出涕者；闻曹操败，即喜唱快”。 $^{③}$ 

“复有俳优旃孟，滑稽淳于，诙谐方朔，调笑酒胡，纵横谑浪，突梯嗫嚅，混妍丑于戚施变舒惨于蘧篋”。 $^{④}$ 

以上材料记载：娱、笑、喜、诙谐、看耍、唱快、滑稽等词语，都围绕一个中心意思即快乐、消闲、娱乐等，可知勾栏瓦子为闲暇娱乐之所。

勾栏瓦子的大众娱乐性形成了市民的享乐风尚，普及到北宋东京城之外的大中城市都崇尚娱乐消遣。《鸡肋编》载：“成都自上元至四月十八日，游赏几无虚辰。使宅后圃名西园，春时纵人行乐。初开园日，酒坊两户各求优人之善者，较艺于府会。……自旦至暮，唯杂戏一色。坐于阅武场，环庭皆府官宅看棚。棚外始作高凳，庶民男左女右，立于其上如山”。 $^{①}$ 同书还记载其他城市如：邓州、宁州、襄阳等地区，节日期间，节目精彩纷呈，呈现万民同乐的场面。

大众娱乐性也是勾栏瓦子得以于长久发展的主要原因之一，其大众娱乐性可概括为：勾栏瓦子是普通大众的闲暇消遣之地，瓦子的娱乐消遣，融赏、饮、赌、嫖、玩等感官享乐为一体，集视、听、味、嗅、触生理快感于一身，使观众获得全方位的娱乐和享受。勾栏瓦子的这种大众娱乐性影响深远，形成了宋代瓦子文化的独特特征，也是宋代社会唯有的文化现象。

# 2、分散性

勾栏瓦子的分散性表现在：勾栏瓦子位置的分散，瓦子艺人和演出内容的分散，以及由此产生了各类分布广泛的休闲娱乐场所，满足了不同地区、不同时间、不同职业观众的娱乐需求，使勾栏瓦子的演出活动遍及到社会的各个角落。

北宋东京的除了固定勾栏瓦子之外，各类流动的勾栏、影戏棚子等也分布在东京城内外，除此之外东京的各类酒楼、妓馆、茶坊、妓馆、寺庙以及大街小巷都有瓦子艺人的身影。如史书记载：

“街市有乐人三五为对，擎一二女童舞旋，唱小词，专沿街赶趁。元夕放灯、三春园馆赏玩。及游湖看潮之时，或于酒楼，或花街柳巷妓馆家祗应，但犒钱亦不多，谓之‘荒鼓板’”。 $^{①}$ 

“且谓杂剧中末泥为长，每一场四人或五人”。②

“如北瓦、羊棚楼等，谓之‘游棚’。外又有勾栏甚多，北瓦内勾栏十三座最盛。或有路岐，不入勾栏，只在耍闹宽阔之处作场者，谓之‘打野呵’，此又艺之次者”。 $^{③}$ 

“此外如执政府墙下空地。诸色路岐人，在此作场，尤为骈阗。又皇城司马道亦然。候潮门外殿司教场，夏月亦有绝伎作场。其它街市，如此空隙地段，多有作场之人”。 $^{④}$ 

“斜阳古柳赵家庄，负鼓盲翁正作场。死后是非谁管得？满村听说蔡中郎”。③

以上材料显示：瓦子艺人表演节目的数量有三五为对、四人、五人等人数不等，演出场地除了勾栏瓦子之外，其他如酒馆、妓馆、酒楼、茶肆、街市的空隙地段等，也是瓦子艺人经常流动演出的地方。由于演员人数和演出场地限制较小，演出活动具有灵活性和流动性，能短时间的吸引观众，也吸引了更多的人加入到这个行业。但是不利于对演出质量及演出单位的管理控制，短时间内无法提高演员素质和演出质量，不宜于勾栏瓦子的长久发展，也无法形成良好的商业演出环境。

# 3、专业性

勾栏瓦子的专业性首先表现在：表演人员及表演团体的专业性。

勾栏瓦子的发展，培养和造就了大批从事专业演出活动的艺人，他们技艺超群，名声鹊起，深受市民喜爱。瓦子艺人的专业性，得源于师傅的严格要求和徒弟的刻苦训练，徒弟时时向老师和同行请教学习，具有“学而不厌”的精神，老师教徒授业则更要“诲人不倦”，就难怪瓦子艺人有过硬的专业水平了，如：

“观者夹路，鼓吹振作。教坊使丁仙现舞，望仁宗御像引袖障面，若挥泪者。都人父老皆泣下”。 $^{①}$ 

“孝宗奉太皇寿，一时御前应制多女流也……演史为张氏、宋氏、陈氏，说经为陆妙慧、妙静，小说为史惠英，队戏为李瑞娘，影戏为王润卿，皆中一时慧黠之选”。 $^{②}$ 

“江浙间路岐伶女，有慧黠知文墨，能于席上指物题咏，应命辄成者，谓之合生。其滑稽含玩讽者谓之乔合生。盖京都遗风也”。 $^{③}$ 

“说唱诸宫调，昨汴京有孔三传编成传奇灵怪，入曲说唱；今杭城有女流熊保保及后辈女童皆效此，说唱亦精。” $^{④}$ 

“李总正、张哥、薛子大、薛子小、曹保义、赵七、丁仪、孔三传、毛详、霍百丑等众艺人”。 $^{⑤}$ 

有材料①②可知，演员的表演绘声绘色，身心投入，如同亲临，使人们感动的声泪俱下，极具感染力。材料②③④则记载了不同表演节目的优秀表演者，并在市民中间留下了很好口碑。

令瓦子艺人值得骄傲的是成为职业艺人，进入宫廷献艺，如丁现仙、徐宣教、李郎中、周八官人、王六大夫等都是御用艺人。另据《宋元杂艺考》记载艺人的名字如：乔万卷、张解元、张贡士等，“解元”、“贡士”是科举的功名名称，“万卷”取自于“读书破万卷”，据名字推测，可知他们的文学功底是相当好的。深厚的文学功底和刻苦的训练决定了瓦子艺人高超的演出水平，他们中间有的博览群书、博古通今，有的琴棋书画样样精通，有的诙谐幽默，滑稽搞笑，受观众喜爱，对后来戏剧、杂技、歌舞、评书、小说等艺术形式的发展影响深远。

勾栏瓦子演员和演出团体的专业性，培养了大批为之奋斗的从业者和爱好者，至南宋时期形成了专门的娱乐消遣行业，并得到了空前的发展和繁荣。其中“社会”就是他们的行业协会组织，是他们交流经验、讨论演出心得的场所，从而促进演出质量提高和演员素质的培养。《西湖老人繁胜录》、《武林旧事》卷三、《梦粱录》卷十九、《都城纪胜》“社会”都记载了社会的发展盛况，主要有唱赚的遏云社、相扑的角抵社、小说的雄辩社、吟叫的律华社、杂剧绯绿社、射弩锦标社、使棒英略社、耍词同文社、影戏绘革社、清乐清音社等。

行会与行会之间、书会成员之间，为了提高各自的实力，不定期的进行交流沟通，这种交流通常在茶坊进行，如《都城记胜》“茶坊”条记载：“茶楼多有都人子弟占此会聚，习学乐器，或唱叫之类，谓之‘挂牌儿’……又有一等专是娼妓弟兄打聚处；又有一等专是诸行借工卖伎人会聚行老处，谓之市头”。 $^{①}$ 由此可知：学习乐器、唱叫等伎艺人定期在茶楼里进行专业学习，交流经验，协调各行会之间关系。

其次，表演内容的专业性

表演内容的专业性，表现在节日期间或重大庆典活动场合，瓦子艺人和教坊钧容直同台献艺时，各节目之间的衔接、配合所表现出来的成熟性、连贯性罕为少见，在当时是很先进的，其盛况可见《东京梦华录》卷九记载：

“……乐未作，集英殿山楼上教坊乐人效百禽鸣，内外肃然，止闻半空和鸣，若鸾凤翔集……。第一盏御酒，歌板色，一名‘唱中腔’，一遍讫，先笙与萧笛各一管和，又一遍，众乐齐举，独闻歌者之声 宰臣酒，乐部起倾杯。百官酒，三台舞旋，多是雷中庆。其余乐人舞者，诨裹宽衫，唯中庆有官，故展裹。舞曲破擷前一遍。舞者入场，至歇拍，续一人入场，对舞数拍。前舞者退，独后舞者终其曲，谓之‘舞末’。第二盏御酒，歌板色，唱如前。宰臣酒，慢曲子。百官酒，三台舞如前。第三盏左右军百戏入场，一时呈拽。所谓左右军，乃京师坊市两厢也，非诸军之军。百戏乃上竿、跳索、倒立、折腰、弄盏注、踢瓶、筋斗、擎戴之类，即不用狮豹大旗神鬼也。艺人或男或女，皆红巾彩服。殿前自有石镌柱巢，百戏入场，旋立其戏竿。凡御宴至第三盏，方有下酒肉、第四盏、第五盏、第六盏、第七盏、第八盏……第九盏御酒慢曲子，宰臣酒慢曲子，百官酒三台舞。曲如前。左右军相扑。下酒：水饭，簇钉下饭。驾兴。……宴退，臣僚皆簪花归私第，呵引从人皆簪花并破管钱。诸女童队出右掖门，少年好俊，争以宝具供送，饮食酒果迎接，各乘骏骑而归。或花冠、或作男子结束，自御街驰聚，竟呈华丽，观者如堵。省宴亦如此”。①

以上材料显示：北宋时期娱乐节目丰富，且每个节目之间的协调性、整齐性、完整性、系统性无可挑剔，这些表演者中间也不乏勾栏瓦子艺人，他们表现出的敬业精神和专业水准是很难得的。

其他勾栏瓦子表演节目，如影戏、说话、相扑、唱赚、小唱等等，这些表演节目内容丰富、种类繁多、选材广泛、生动感人，令广大观众流连忘返。

最后，表演道具的专业性

表演道具的专业性，主要是一些特殊的节目，如傩戏、杂技、舞蹈等。据史书记载：“政和中大傩，下桂府进面具，比进到，称‘二副’。初讶其少，乃是以八百枚为一副，老少妍陋无一相似者，乃大惊。至今桂府做此者，皆致富，天下及外夷皆不能及”。 $^{①}$ 材料显示：傩戏的面具，桂府（今广西）做得最好，精致细腻、惟妙惟肖，不管男女、老少、美丽、丑陋没有一个相似，可见制作技艺的高超。

服装的专业性。据《东京梦华录》卷七记载：“女童皆妙龄翘楚，结束如男子，短顶头巾，各着杂色锦绣捻金番丝缎窄袍，红绿吊敦束带。莫非玉鞍金勒，宝鐙花靴，艳色耀日，香风袭人”。②另《都城纪胜·瓦舍众伎》记载：“其诸部分紫、绯、绿三等宽衫，两下各垂黄义襕。杂剧部又戴诨裹，其余只是帽子幞头”。③由材料可知表演的服装有：面具、青色花衫、衣衫（紫衫、宽衫）、浑裹、吊敦、幞头（卷角幞头）、窄袍、宽袍、帽子（金华小帽、小隐士帽）、头巾、簪花，锦绣鞋子、鲜花等，“吊敦”据刘念慈考证“是袜裤相连的装束”。④服装代表了不同的人物角色。这种用服装区分角色的做法，便于区分行业艺人和不同的角色分工，便于观众和市民识记。他们的服装可分为：袍子、衣衫、裤子、帽子、头饰、鞋子、衣饰等等，加之其他各种附加道具，增添了表演的氛围，突出了表演的特色。

# 4、商业性

勾栏瓦子作为固定的演出场所，首开商业性演出活动的先例，使大批民间艺人有了用武之地，他们终日在勾栏演出，为文艺活动的繁荣提供了舞台和空间。据史书记载：

“白秀英拿起盘子指着道：‘财门上起，利地上住，吉地上过，旺地上行，

手到的面前，休教空过。'白玉乔道：‘我儿且走一遭，看官都带赏你’”。①

“见一个人手撑着橡做的门”，“要了二百钱放过咱”。 $^{②}$ 

“……小雪持过红丝盘子，下台便问观众索缠头钱。豪客、官家、各争赏赐”。③

资料中“手到面前，休叫空过”、“看官都带赏你”、“豪客，官家，各争赏赐”描写的是表演节目结束时，演员向观众索取报酬的场面。以此可知，除了在观众在进入瓦子时需要付费之外，观看演出也另外付费，只是数量不限，全凭观众意愿，这使得瓦子艺人的收入不等。勾栏瓦子为大批艺人提供了就业机会，使他们有了自力更生的谋生手段，并由此可能过上富裕的生活。如《明道杂志》载：“……嘱弄者，且缓之。……请既而祭之。此子闻甚喜，弄者乃求酒肉之费，此子出银器数十”。④材料记载的是一瓦子艺人的表演赢得观众的认可，并给予“银器数十”的赏钱。又如《砚北杂志》载：“温饱逍遥八十馀，稗官元自汉虞初。世间怪事皆能说，天下鸿儒有弗如”，⑤记载了南宋瓦子艺人王大夫的富裕生活。由此可知，勾栏瓦子的商业性促使很多人自愿的加入到这个行业里来。

随着勾栏瓦子的不断发展，瓦子艺人之间、瓦子之间都存在激烈的商业竞争，为了生存他们不断提高演出水平、丰富演出内容和加强对演员基本功的训练。据吴晟研究证明：“《元明杂剧·蓝采和》第二折：【梁州第七】咱咱咱，但去处争利夺名，若逢对棚，怎生来妆点的排场盛？依仗这书会社恩官求些好本领。这里“但去处争夺名利”、“对棚”指的就是两个戏班对台演出时，各演出单位为了生存而争夺名利，碰上这种情况，就得凭艺人的真本事。又如：《宦门子第错立身》第五出：（生）闲话且休提，你把这时兴的传奇，（旦白）看掌记。（生连唱）你从头与我再温习”。①由此可看出：想在勾栏里面站稳脚跟，不仅要求演员对表演内容烂熟于口，而且还要尽可能多的开发新剧目，尤其是当前的热点话题。为了在竞争中立于不败之地，各演出团体想尽一切办法来提高知名度和演员素质。

# （二）勾栏瓦子消费群体分析

# 1、官僚及士大夫群体

北宋东京勾栏瓦子是闲暇时间的娱乐之地，对人们生活影响极大，勾栏瓦子的消费群体遍及社会的各个阶层。其中最具消费能力的是官僚、富家子弟及士大夫阶层，他们拥有较多的闲暇时间和经济实力，是瓦子里面主要消费群体之一，可见有关的史料记载：

“遇雪，公子王孙赏雪，多乘马披毡笠，人从油绢衣……深冬冷月无社火看，却于瓦市消遣”。 $^{②}$ 

“顷者京师甚为士庶放荡不羁之所，亦为子弟流连破坏之门。……。今贵家子弟郎君，因此荡游，破坏尤甚于汴都也”。 $^{③}$ 

“临安中瓦在御街中，士大夫必游之地，天下术士皆聚焉”。④

“士人便服日至瓦市观优，有邻坐者，士人与语颇狎”。 $^{③}$ 

“……小雪持过红丝盘子，下台便问观众索缠头钱。豪客、官家、各争赏赐”。⑥

从资料中的：“公子王孙”、“贵家子弟”、“豪客”、“官家”、“豪贵子弟”等可以看出，官家子弟、富家子弟都是勾栏瓦子的常客，他们经济条件优越，闲暇时间多，就经常光顾勾栏瓦子。从资料①②可以看出“士大夫”、“术士”、“士人”等文人士大夫阶层，业余时间也经常在勾栏瓦子内欣赏歌舞、品茶、喝酒、聊天等。由此可见，勾栏瓦子的消费群体可概括为：富家子弟、士大夫、官员子弟，他们物质条件优越，所给“费用”较高，是瓦子里面重点争取的消费群体。

# 2、军士及其家属

北宋开国皇帝赵匡胤，采用“杯酒释兵权”的谋略，将建国功臣妥善安置，上曰：‘人生如白驹过隙耳，所谓富贵者，不过欲多积金钱，厚自娱乐，使子孙显荣耳。汝曹何不释去兵权，择便好田宅市之，为子孙立永久之业，多置歌儿舞女，日饮食相欢以终天命。君臣之间两无猜嫌，上下相安，不亦善乎！ $^{①}$ 使大批武将卸甲归田，并给予他们丰厚的物质待遇，让他们多置田地以立子孙永久之基业，终日歌舞儿女，君臣之间两无猜嫌，岂不上下相安。除此之外，北宋东京收天下甲兵数十万于京师，他们优越的物质待遇，充足的闲暇时间，诸多娱乐场所就成了他们经常光顾之地。

《东京梦华录》“马行街铺席”条载：“州北瓦子，新封丘门大街两边民户铺席外，与诸班直军营相对，至门约十里余……”， $^{②}$ 州北瓦子且对面不远就是军营，两地相距大约十里，良好的地理位置，极大的方便了军士光顾这些场所。同书“诸色杂卖”条载：“或军营停放，乐人动鼓乐于空闲，就坊巷引小儿妇女观看”， $^{③}$ 这里的“小儿妇女”指的驻军家属。《东坡志林》卷1《怀古》载：“至说三国事，……彭恺之子为武吏，颇知文章，余尝为做哀辞，字大牛”。 $^{④}$ 资料记载彭恺之子为武吏，乃属于军营之文吏。

另《梦粱录》卷十九也记载：“杭城绍兴闲驻跸于此，殿岩杨和王因军士多西北人，是以城内外创立瓦舍，招集妓乐，以为军卒暇日娱戏之地”。 $^{①}$ 可见，驻军军士在勾栏瓦子消遣娱乐的风气一直延续到南宋。

由以上分析可知，驻军军士及其家属是勾栏瓦子的观众来源之一。

# 3、普通大众

除了以上官僚及其子弟、士大夫、驻军军士和家属之外，普通大众是勾栏瓦子最大比例的观众来源。如：市内妇女儿童、来京士人、农民、僧人、道士、社会闲散人员、官家家奴、官吏、普通市民等都是勾栏瓦子的观众。

“儿女相携看市优，纵谈楚汉割鸿沟。山河不暇为渠惜，听到虞姬直是愁”。①

“平日里只爱去三瓦两舍……都头雷横和李小二到勾栏里来看……如今见在勾栏里，说唱诸般品调。每日有那一般‘打散’，或是戏舞，或是吹弹，或是歌唱，赚得那人山人海的家看”。 $^{③}$ 

“以布衣游场屋三十年，未成名。在京师，殊无聊，……又游相国寺，与众书生倚殿柱观倡优”。 $^{④}$ 

“至元壬寅夏，松江府前勾栏邻居顾百一者，……。有女官奴，习讴唱，每闻勾栏鼓鸣则入。是日入未几，棚屋拉然有声，众惊散。即而无恙，复集焉。不移时，棚山压。顾走入抱其女，不谓女已出矣，遂毙死于颠木之下。死者凡四十二人，内有一僧人，二道士，独歌儿天生秀全家不损一人。其死者碎首折肋，断筋遗髓”。 $^{③}$ 

“京师有富家子，少孤专财，群无赖百方诱导之，而此子甚好看弄影戏，每弄至斩关羽，为之泣下”。 $^{①}$ 

以上材料证明了普通市民阶层在勾栏瓦子内观看表演的情形，另据洪迈《容斋三笔》记载了北宋开封府，有个农民淳泽观看了艺人表演之后，回家途中遇桶匠便取桶戴在自己头上，以装扮刘备。《庄家不识勾栏》描写的是农民进城逛勾栏的事情。可见，农民也是勾栏瓦子的常客之一。

由以上材料分析可知，勾栏瓦子的消费群体遍及社会的各个阶层，其中官僚、富家子弟及士大夫和驻军军士及家属，属于经济富裕类群体，他们闲暇时间多，但数量有限。普通大众虽然经济水平有限，但数量众多，他们共同构成了勾栏瓦子的消费群体。

# 四、北宋东京勾栏瓦子的主要社会功能

# （一）市民休闲娱乐场所

中国历来是以农业大国著称的，宋代以前农民的闲暇时间生活相对单调。北宋时期是个例外，随着农村自然经济的凋敝，商品经济的发展，大批农民涌入城市，使城市人口急剧增加，促成了市民阶层的兴起。北宋东京市民阶层已经发展成为社会的主要力量之一，引起社会的广泛关注，他们拥有一定的经济实力，原来的生活方式已无法满足他们的各种需求，勾栏瓦子的诞生，丰富了他们的业余生活，是他们闲暇时间的娱乐之所，成为日常生活中的一个组成部分。

勾栏瓦子为官僚、士大夫和市民阶层提供了闲暇娱乐之地，如《西湖老人繁胜录》所载：“深冬冷月无社火看，却于瓦市消遣”， $^{①}$ 又如史书记载：“士人便服日至瓦市观优，有邻坐者，士人与语颇狎”， $^{②}$ 南宋《梦梁录》卷十九记载，南宋临安的瓦子专门为驻军创设，是他们闲暇娱戏之地。材料证明：观众去勾栏瓦子的主要目的是寻求精神上的愉悦和满足，丰富业余生活。

勾栏瓦子是大中型城市中的休闲娱乐场所，勾栏瓦子做为娱乐场所，休闲娱乐是第一要务，对观众的吸引力主要表现在：勾栏瓦子的表演没有宫廷演出的清规戒律，适合市民口味的大众性娱乐，表演风格活泼、自由、符合普通大众的心理需求。除了表演娱乐活动之外，还经营一些酒楼、茶肆、商贩等，便于瓦子内观众空闲之余与同伴聊天、消遣、娱乐等，是东京市民闲暇时间的主要休闲娱乐场所。勾栏瓦子全年营业，表演活动从早到晚轮流演出，使看客流连忘返，丰富了市民平日里的休闲生活，所谓“终日居此，不觉抵暮”。勾栏瓦子的兴起也引起了北宋东京城市结构发生变化，注重休闲空间的建设，各酒楼、妓馆、茶楼、夜市、集市、各种小吃店等，供市民业余娱乐、消遣、谈生意、朋友聚会、饮食住宿等场所分布广泛，形成了星罗密布的休闲娱乐场所分布网络。

勾栏瓦子的兴盛，促使东京的娱乐活动丰富多彩。尤其是节日期间各娱乐活动更是令市民眼花缭乱。宋代节日主要有清明节、元宵节、中秋节、上元节、冬至、春节等，每逢节日期间官府休假，并放关扑三日，允许市民大肆庆贺，各种娱乐活动数不胜数，勾栏瓦子艺人也争相献艺、表演，大街小巷都有他们优美的歌声和精彩的表演。如《东京梦华录》卷六记载：“正月一日年节，开封府放关扑三日。士庶自早互相庆贺，坊巷以食物动使果实柴炭之类，歌叫关扑”。 $^{①}$ 节日期间允许妇女上街出游，从早上开始，市民穿上干净的、华丽衣服往来拜节，各巷、坊门口等热闹地方摆满了食物、日用品、水果、歌叫关扑等。市民还聚集御街两廊下，观看歌舞、百戏、奇术异能等娱乐活动，东京城各主要街道如：马行街、潘楼街、州东东门外等处，也都搭起彩棚、乐棚等，铺陈冠梳、珠翠、头面、幞头、玩好之物，供人选看，大街小巷都挤满了庆祝节日的市民。

可见，北宋东京市民，娱乐场所丰富，一年四季都可以欣赏到不同的表演娱乐活动，极大的丰富了闲暇业余生活，提高了生活质量。

# （二）演出场所

勾栏瓦子作为固定的演出场所，演出节目涵盖了说话、杂剧、杂技、傀儡戏、木偶戏等等，使民间艺人有了固定的表演空间，也培育了一大批专门的民间表演艺人。他们有些终日在一家勾栏“做场”，如：《西湖老人繁胜录》载：“女流，史惠英、小张四郎，一世只在北瓦，占一座勾栏说话，不曾去别瓦作场，人叫做小张四郎勾栏”。①《蓝采和》杂剧中蓝采和说：“俺在这梁园棚一交却又早（是）二十年”，梁园棚是洛阳著名的勾栏，这些在勾栏瓦子里长期的固定演员，都是勾栏瓦子艺人中的名角，对当时乃至后世都影响极大。

当然，还有一些技艺稍逊的艺人，则只能轮流演出，如《东京梦华录》卷二载：“自丁先现、王团子、张七圣辈，后来可有人于此作场”。②《西湖老人繁胜录》载：“尚有独勾栏瓦市，稍远于茶，中作夜场”，瓦市“常是两座勾栏，专说史书；背做莲花棚，常是御前杂剧”。③由材料可知：这些技术稍逊的瓦子艺人只能轮流做场，常常是按时间先后排列或是两座勾栏轮流演出，这种轮流演出的表演方法，满足了不同层次观众的需求，现在看来在当时是比较合理和先进的。

作为演出场所的勾栏瓦子不受时间限制和天气的影响，如“不以风雨寒暑，白昼通夜，骈阗如此”， $^{④}$ “夜市直至三更尽，才五更又复开张。如耍闹去处，通宵不绝”。 $^{⑤}$ 由此可知，勾栏瓦子已经发展成为比较成熟的商业演出场所，昼夜营业。

北宋时期，能在勾栏瓦子里进行表演的艺人，都是才艺双绝，为了能在勾栏里长期演出，这些瓦子艺人，需要不断的进行演出节目创新和训练，而那些不能在瓦舍里表演的艺人，只能在路边一些宽阔之地进行表演，称之为“打野呵”。

作为演出场所的勾栏瓦子，为勾栏瓦子艺人提供了固定的表演场所，是为宋代首创，这对文艺节目的创新，演员素质的提高以及形成专业商业演出场所有很大的积极作用。演出娱乐场所的兴盛，促使娱乐节目和演艺活动逐渐增多，尤其是娱乐活动，娱乐活动在古代被认为是“玩耍”的活动，历来被认为是玩物丧志的表现，是大多数上层人士所鄙夷的事情，而普通老百姓被生活所迫，很少参与娱乐活动，而宋代却开创了一代“玩耍”之风。

宋代的娱乐活动节目之多、质量之高、参演人员之众历代少见，并且宋代的娱乐活动官府限制较少，甚至提倡。如皇家园林——琼林苑，每年三月一日，向市民开放一个月。届时，勾栏瓦子艺人、教坊艺人争相演出，一些路岐人也在此献艺，形成了百艺呈现的壮观场面。

“三月一日，州西顺天门外开金明池琼林苑，每日教习车驾上池仪馆。虽禁从士庶许纵赏，御史台有榜不得弹劾。……中坐各设御幄，朱漆明金龙床，河间云水，戏龙屏风，不禁游人。殿上下回廊皆关扑钱物饮食伎艺人做场，勾肆罗列左右。桥上两边用瓦盆，内掷头钱，关扑钱物、衣服、动使。游人还往，荷盖相望。……街东皆酒食店舍，博易场户，艺人勾肆，质库，不以几日解下，只至毕池，便典没出卖”。 $^{①}$ 

文中提到开金明池期间，皇帝驾临观看、须士庶观赏、御史台有榜不得弹劾，可知官府对娱乐活动是提倡的，提到的娱乐活动主要有：金明池争标、艺人做场、官扑、百戏、垂钓、饮食、博弈等等，形成了与民同乐、万民共欢的场面。其他《东京梦华录·驾幸临水殿观争标赐宴》里面记载了几十种娱乐节目，如：大旗、狮豹、棹刀、蛮牌、神鬼、杂剧、木偶、水傀儡、水秋千、小龙船表演、飞鱼船、虎头船、鳅鱼船、大龙船等都竞相献艺，使人们眼花缭乱，目不暇接。宋代不仅继承了已有的娱乐活动，开发和新创了新的娱乐形式，北宋东京的娱乐活动节目到底有多少，目前还没有准确的统计，但从历代社会娱乐活动的发展状况来看，是前所未有的。

北宋东京皇官家园林定期向市民开放的娱乐风尚，打破了士庶界限，形成了官民同乐的融洽氛围，增加了市民对娱乐活动的兴趣，这种风尚遍及到北宋的各个地区，如：辽穆宗十八年（968年）正月己亥（十五日元宵），穆宗“观灯于市，以银百两市酒，命群臣亦市酒纵饮三夕”。 $^{①}$ 可见，宋娱乐活动的感染力和延展性。

北宋东京丰富多彩的娱乐活动对现代社会城市市民休闲生活方式的选择有很大的参考价值。

# （三）商品交易场所

勾栏瓦子作为商业交易场所，不仅表现为商业性的演出场所，而且也为无业人员提供了就业机会，他们在勾栏瓦子里面出售货物，以此养家糊口。《东京梦华录》卷二记载：“瓦中多有货药、买卦、喝故衣、探搏、饮食、剃剪、纸画、令曲之类。终日居此，不觉抵暮”。②勾栏瓦子内出售货药、令曲、纸画、剃剪等物品，都是演员和观众的所需之物，其中令曲，据扬宽先生考证，是指出售一般曲谱唱词的摊子。另：“中瓦内还有刊印小说的张家书铺”。③这些多是瓦子内小商贩的经营活动，勾栏瓦子的不断发展，瓦子内的经营品种日益丰富。

南宋时期，勾栏瓦子出现了经营大型的酒楼、茶坊、起店等。据《西湖老人繁胜录》记载：北瓦“内有起店数家，大店每日使猪十口，只不用头蹄血脏。遇晚烧晃灯拔刀，饶皮骨，壮汉只吃得三十八钱，其吃不了皮骨，饶荷叶裹归，缘物贱之故。其每袋七十省，二斤二两；肉卖九十省，一斤”。①分析材料可知，起店可能是餐馆，主要经营日常饭菜，满足了勾栏瓦子艺人和观众的日常饮食所需。

《梦梁录》卷16记载：“中瓦内王妈妈家茶肆名“一窟鬼茶坊”， $^{②}$ 这是中瓦内著名的茶坊。同书又载：“次有南瓦子熙春楼王厨开沽”， $^{③}$ 对南瓦子与熙春楼的位置学界存有争议，有学者认为熙春楼和南瓦子是并列的，还有学者认为是熙春楼在南瓦子里，本人认为：根据字面意思和语言习惯理解应为：南瓦子里面的熙春楼，因为，在北宋时期瓦子盛行，茶肆和起店在瓦子内经营，以此也可以推断，酒楼也很有可能在瓦子内。另外，瓦子全天营业，观众需要吃饭休息的场所，而酒楼是日常饮食必须之地，所以可以认为，熙春楼是在南瓦子里面经营的。

从以上相关材料分析，北宋勾栏瓦子不仅是一个娱乐表演场所，而且还是一个集饮食、娱乐、商业交易等休闲场所，集综合性、商业性、娱乐性为一体的大型休闲娱乐消费场所。

# （四）促进东京城市功能的转变

北宋东京政局稳定，文化繁荣，经济发展，形成了市民善于享乐，懂得享受的生活方式。为了满足这种需求，东京城内外形成了以勾栏瓦子为中心，以酒楼、妓馆、茶肆、寺院道观园林、官私家园林、郊外绿地为补充的公共休闲娱乐场所，遍布各个角落满足了不同地区的市民娱乐需求。

《东京梦华录》多处记载了记载东京城街道两旁的酒楼、妓馆、茶肆、瓦子等休闲娱乐场所和饮食场所，由以下材料可以看出：

“则旧曹门街，北山子茶坊，内有仙洞、仙桥，仕女往往夜游，吃茶于彼。……。出旧曹门，朱家桥瓦子。下桥，男斜街、北斜街，内有泰山庙，两街有妓馆。……看牛楼酒店，亦有妓馆，一直抵新城。自土市子南去铁屑楼酒店、皇建院街、得胜桥郑家油饼店，动二十余炉，直南抵太庙街、高阳正店，夜市尤盛。土市北去，乃马行街也，人烟浩闹。先至十字街，曰鶉儿市，向东曰东鸡儿巷，向西曰西鸡儿巷，皆妓馆所居。近北街曰杨楼街，东曰庄楼，今改作和乐楼，楼下乃卖马市也。近北曰任店，今改作欣乐楼，对门马铛家羹店”。①

# 五、北宋东京勾栏瓦子的流变

# （一）演出场所的裂变

勾栏瓦子作为固定的表演场所是宋代首创的，发展至南宋时期，规模日益加大，数量更多，但是，入元以后，其局限性越来越突出，加之朝代的更迭和人们生活水平的改变，使勾栏瓦子这独有的文艺演出空间，不断地发展变化，蜕变为不同的演出场所。

勾栏瓦子历经时代的变迁，逐步的演变为不同的娱乐场所。据陈雪飞研究证明：古代勾栏为三面敞开的伸出式戏台，与现代农村戏楼和城市舞蹈剧场有相似之处，可以说是由勾栏瓦子直接转化而来的。梁俊雄、徐剑锋、梁小珍合写的《宋明“瓦舍”与“健身娱乐场所”社会功能探析》一文研究证实：瓦子与现代健身娱乐场所在社会功能方面的相似之处，即都有消遣娱乐、商业交易、社会交际的功能，并且现代建设娱乐场所也符合勾栏瓦子的商业交易性、娱乐性、大众性等特点。因此，可以说现代健身娱乐场所是瓦子功能的变迁。

勾栏瓦子作为古代最早的综合性休闲娱乐演出场所，集吃、喝、玩、娱、表演等于一体，这种具有多重演艺功能的休闲娱乐场所，为音乐、舞蹈、戏剧、相声、魔术、小品、评书、杂技等文艺表演活动提供了广阔的发展空间，成就了艺术的繁荣。根据本文对勾栏瓦子的特征和主要社会功能的研究分析，可以说勾栏瓦子是现代休闲娱乐场所和表演场所的前身，为休闲娱乐场所和休闲娱乐活动的发展提供了历史参考依据。

# （二）“说话”促使了小说的诞生

小说在唐以前出自于宫廷文人雅士之手，鲁迅先生被称为是“叙述宛转、文辞华丽” $^{①}$ 之作，而宋代则大改这种浮华、绮丽的文风，向着平民化、世俗化发展，据史书记载：“小说起宋仁宗，盖时太平盛久，国家闲暇，日欲进一奇怪之事以娱之，故小说得胜”， $^{②}$ “中瓦内还有刊印小说的张家书铺。” $^{③}$ 鲁迅先生也曾说过：“宋初修的《太平广记》为小说之渊薮”， $^{④}$ 并对宋代小说进行评论言：“宋一代文人之为志怪，既平实而乏文彩，其传奇，又多托往事而避近闻，拟古且远不逮，更无独创之可言矣。然在市井间，则别有艺文兴起，即以俚语著书，叙述故事，谓之‘平话’，即今所谓‘白话小说’者是也”。 $^{⑤}$ 有关小说是否是勾栏瓦子说话艺人的底稿，学界存有争议，二十世纪八十年代，中国大陆出版小说史、文化史都将话本理解为勾栏瓦子艺人说话采用的话本。鲁迅则提出，元明以后文人模仿勾栏瓦子艺人话本而创作的小说题材，鲁迅称之为“拟话本”，并且被大多数学者接受，可以说是今日之小说。但在1965年，日本学者增田涉提出了异议，认为话本不是说话人用的底稿，而是有故事的意义，并且用大量例子来证明自己的论点，也得到了学术界多数人的肯定。傅承洲教授又提出了艺人话本和文人话本概念。

前面所说的拟话本、艺人话本、文人话本，都可以说是现代小说之渊薮。因此，可以说小说萌芽于宋以前，兴起于北宋时期，经过宋元的发展创作，明清时期逐渐成熟并达到高峰，使小说作为专门的文学创作形式固定下来，并产生了大量的小说家和小说作品，如四大名著《水浒传》、《三国演义》、《红楼梦》、《西游记》是我国古代小说的巅峰之作被人们广泛称颂。

# （三）杂剧流变为现代戏剧

北宋杂剧是勾栏瓦子里面主要演出活动之一，它是在唐朝参军戏的基础上发展而来的，经过在瓦子里面的千锤百炼，最终发展为现代戏剧。然而，现代戏剧的形成不是一朝一夕的事情，是在历代宫廷艺术和市民文艺的融合中孕育而来的，北宋东京勾栏瓦子是促成戏剧产生、发展，并最终成为现代之戏剧的温床。

关于杂剧与戏剧的传承关系，很多资料都有记载。宋人耐得翁《都城纪胜》“瓦舍众伎”条云：“杂剧中，末泥为长，每四人或五人为一场，先做寻常熟事一段，名曰艳段；次做正杂剧，通名为两段。末泥色主张，引戏色分付，副净色发乔，副末色打诨，又或添一人装孤。其吹曲破断送者，谓之把色。大抵全以故事世务为滑稽，本是鉴戒，或隐为谏诤也，故从便跣露，谓之无过虫”。 $^{①}$ 所谓“艳段”，就是开场引子的意思，先演出一段比较简单的杂剧段子即“囊”，然后副净带领其它角色一起登场，大家共同念诗诵词，插科打浑，完毕退下，正戏开始。根据末泥色、引戏色、副净色、副末色名称来看，已经具备了戏剧的基本特点，即有一定的人数和角色分工，并且以讲述一个完整的故事为主，符合现代戏剧的特征。

元人陶宗仪《南村辍耕录》卷25记载：“唐有传奇，宋有戏曲，唱诨，词说。金有院本、杂剧、诸宫调。……院本、杂剧，其实一也。院本则五人，一曰副净，古谓之参军。一曰副末，古谓之苍鹘，鹘能击禽鸟，末可打副净，故云。一曰引戏。一曰末泥。一曰孤装，又谓之五花爨 cuan弄。或曰宋徽宗见□国人来朝，衣装鞋履巾裹，传粉墨，举动如此，使优人效之以为戏”。①由此可以看出，元代的时候，杂剧表演已有明确的角色分工，净、旦、末、丑、泥五种角色已经形成。

元代戏剧除了具备角色分工之外，面部化装和服装也逐渐完善。首先是它的滑稽角色继承了宋金杂剧中副净、副末角色用墨道贯眼、黑蝴操遮面的画法，以及宋代南戏里净角和丑角的粉墨涂面。从出土文物来看，山西省丙城县元潘德冲墓石棺杂剧线刻画、运城西里庄元墓杂剧壁画中，那两位净角仍然用黑墨道贯穿眼眉上下，新绛县寨里村元墓杂剧砖雕左侧第二人颜面上仍然用墨画黑蝴蝶，洪洞县明应王殿元杂剧壁画里，前排左侧第二个净角用白粉涂成白眼圈、白嘴圈，并涂红嘴唇。这是戏剧脸谱和服装形成过程的实物资料。由此可知，元代时期，戏剧已经有了服装、脸谱、角色，加速了现代戏剧的形成。

吴晟《瓦舍文化与宋元戏剧》一文提出宋元剧种主要有：目连戏、宋杂剧、金院本、戏文、元杂剧、傀儡戏、影戏。其中，戏文和元杂剧已是成熟的戏剧形式。吴晟认为宋元戏剧在一定程度上都来源于北宋东京勾栏瓦子里的文艺创作，并进一步推断：宋徽宗时期，已出现“戏文”，并将“真戏剧”的产生归纳为四个条件：一、曲本位的确立；二、角色行当体制的形成；三、唱念做打四功的融合；四、有一定长度的剧情结构。而这些特征的形成时间是北宋，地点就是勾栏瓦子，正是那些艺人孜孜不倦的努力才创造了现代戏剧的辉煌成就。据吴晟考证：最早的戏曲节目是勾栏瓦子表演的杂剧《目连救母》，它是瓦子内多种伎艺融合的结果，曾创下了演出时间最长、观众最多的记录。景建军《勾栏瓦舍与中国戏曲的一元化视界结构》一文，也论证了瓦子内的杂剧、诸宫调、说话等艺术形式相互融合，共同孕育并最终促使了中国戏曲的形成。

经过无数勾栏瓦子艺人和元明以后艺人的不断努力和改造，使宋元杂剧这一古老的民间艺术形式，开出了灿烂的现代戏剧艺术之花。如今，我国现代戏剧在国际上享有盛名，尤其是京剧被称为我国的“国粹”，其它的戏剧派别也竞相发展，如粤剧、川剧、晋剧、豫剧、昆曲等，昆曲还被列为“人类口述遗产和非物质遗产”。今日戏剧的发展大放异彩，各戏剧派别和唱腔丰富了我国的戏剧艺术宝库，为中国戏剧艺术的发展做出了巨大的贡献，而这一切得源于北宋勾栏瓦子这个演出场所，为艺术的发展提供了平台，使之在这里诞生、成长。

# （四）绝活、杂技表演水平日益提高

杂技萌芽于春秋战国以前，一般认为春秋时期已出现完整的杂技形式，《列子·说符篇》记载周穆王西游，途中遇到能工巧匠偃师献歌舞假人，如同真人，能摇头晃脑、唱歌、跳舞，跳舞结束，居然与周穆王的妻妾调情。周穆王大怒，偃师立即将假人拆开，发现竟是用木头做成的，这就是最早的有关杂技木偶戏的记载。汉代是中国杂技的成熟期，同时汉朝与各国的交流加强，丰富了杂技的内容，直到汉武帝时期，杂技百戏成为一种宫廷艺术，专门在宫廷内演出。北宋时期，勾栏瓦子使杂技百戏走向人民大众，成为了一种大众民间艺术，元明清以后，杂剧发展慢慢的走入低谷，成为不登大雅之堂的艺术，只能在农村和边缘地带表演。

因杂技被认为是“地摊”艺术，只适合勾栏瓦子演出和江湖卖艺，致使现代杂技发展陷入“不发展”的尴尬境地，致使从事杂技艺术的人越来越少。然而，经过上千年的发展，中国现代杂技与宋代相比不可同日而语，现代杂技千变万化，种类齐全，技术水平日益提高，节目更加丰富，内容越来越多，动作难度也越来越大，主要可分为：力技、形体技巧类、耍弄技巧类、高空节目、以奇妙变化为表演特征的魔术节目、象形象声节目、驯驭兽畜、滑稽及小丑艺术等等。虽然中国的杂技艺术在国际舞台上频频亮相，日益受到外国人的认可，但是在国内并没有得到足够的重视，认为这是一种没有品味和低俗的表演，很少有人专门买票去看杂技表演，而导致我国著名的杂技表演都是为外国旅游团演出的，也使很多杂技表演艺人到国外演出，这种“国内不香国外香”的场面，是很难长久的。原中国杂技团团长李恩杰曾言：“我从来不认为中国杂技是墙里开花墙外香。我们的民族艺术如果在本土开花都不香，到外面也香不了。中国杂技出国演出不少，但多数停留在低端市场，无法形成自己的品牌。我们只有下大力气把国内市场做好，才能在国际上拥有自己的市场，获得应有的回报。”这说明中国已经充分认识到杂技艺术所具有的魅力，它应该在中国的大地上，开出更加灿烂的果实。

# （五）专业演员队伍形成

北宋东京的勾栏瓦子艺人，行走在城市的大街小巷、酒楼、茶肆、妓馆、寺庙、道观等场所，为丰富城市生活做出了巨大的贡献，他们像影视明星一样，非常受人欢迎，节日期间更是艺人们大放异彩的时候，各临街热闹之处都有他们精彩表演。他们为辛苦工作一天的劳动者，提供了放松休闲的机会，在勾栏瓦子演出是对他们艺术水平和社会知名度最好的诠释。

北宋东京表演水平较高的勾栏瓦子艺人一生在一个勾栏瓦子内演出，而技术稍逊的只能在瓦子以外宽阔热闹之处表演，不同的表演场所造就了不同的瓦子艺人命运。如《东京梦华录》里记载的，小张四郎、王六防御、西山一窟鬼等，一世只在一个勾栏做场；有的演员则只能流动跑场，打野呵，甚至下到农村去卖艺，成为流动的“路岐人”，所谓“斜阳古柳赵家庄，负鼓盲翁正作场”， $^{①}$ 就是流动艺人“打野呵”的表演；有的则名价很高，如一代名妓李师师“然名娼深藏邃阁，未易招呼”， $^{②}$ 其身价和名气都是首屈一指的；有的则在府衙庭院内演出，像小唱名角萧婆婆是韩太师府的歌伎，属于是私人宅邸艺人；有的演员在军营中演出，小说伎艺人王辩就是军营中的“铁衣亲兵”；有的演员还在宫廷演出，如小说伎艺人孙奇、方瑞、任辩、刘和等都曾在皇宫中供职，名望地位很高。可见，北宋东京各种伎艺人已发展成为专门的演出人员。

南宋时期，勾栏瓦子艺人就形成了庞大的专业表演队伍。《武林旧事》就记录了南宋杭州的诸色伎艺及其艺人人数见图五。

如今，随着社会的发展、科技的进步、生活水平的提高，促使人们更加注重精神生活的享受，而文艺演出带来的视觉、听觉享受正好满足了这种需求。这就为文艺演出带来了巨大的发展契机，致使形成了包括音乐、戏剧、小品、电影、电视剧、民间歌曲等等，受过专门训练的各种文艺工作人员。

# 结 语

北宋东京闲暇时间增多，政治氛围宽松，坊市结构转变，商品经济发达，以及市民对休闲场所的需求，所有这些因素共同孕育了勾栏瓦子的诞生。勾栏瓦子在历史长河中虽然是昙花一现，但是它所散发的光芒是耀眼的，不论是当时，还是现在都意义深远。

勾栏瓦子作为固定的娱乐场所，开创了商业性演出活动的先例，其本身的经营管理、社会功能、娱乐活动、发展特征等，在当时都是很先进的。它使北宋东京形成了以勾栏瓦子为中心，以酒楼、茶坊、妓馆、寺院、官私园林相互补充的各类休闲场所分布网络，出现了以满足人民的娱乐需求而提供的饮食、住宿等配套设施设备的服务部门。北宋东京各类休闲娱乐场所的繁荣发展，促使了服务业和文化产业的空前繁荣，增加了就业，稳定了社会秩序，使北宋东京成为当时最繁荣富庶的城市。勾栏瓦子经过历代发展，逐步蜕变成了门类齐全的专门的休闲娱乐场所和演出场所，如剧院、健身场所、茶馆、酒楼等。

勾栏瓦子使无数宫廷文艺节目走向大众，成就了民间艺术的繁荣，它对文艺演出和娱乐活动的影响，是其它朝代其他的文化现象所不能比拟的，它促使了无数文艺节目的诞生、发展，并日益丰富多彩，造就了文艺艺术的繁荣，丰富了现代社会的文艺演出形式。如现代的民间歌曲、戏剧、相声小品、文艺小说、歌舞、杂技、绝活、魔术杂技等等，都得源于勾栏瓦子这一演出平台的影响。

勾栏瓦子作为中国最早的固定演出场所，使民间艺人有了自由的表演空间，成就了无数民间表演艺人，直到现代社会形成了专业的门类齐全的文艺演出人员，中国大众文艺的繁荣发展与无数无名艺人在默默奉献是

分不开的。

勾栏瓦子是成就文艺演出活动、演艺人员和演出场所的平台。勾栏瓦子开创了追求精神享受的时代，各种娱乐活动空前繁荣，固定的演出场所，专业的演员队伍，商业性的演出活动，相关的配套设施设备，共同促进了现代休闲娱乐活动和休闲娱乐场所的诞生。

现在，距离宋代已经有1000多年的历史，逝去的是历史的繁荣与沧桑，留下的是无尽的哲思。勾栏瓦子作为一种文化现象，它所起到过的作用是无法抹去的，它对中国文化与艺术发展的影响都是深远的。然而，随着社会经济的发展，外来文化的涌入，某些传统表演节目现在看来显得有些笨拙、粗糙，逐渐的被遗忘，如皮影戏、傀儡戏、木偶戏、喷火、杂技、评书等，还有一些如乔合生、药发傀儡等可能已经失传。而这些已经消失或正在消失的传统节目或演出活动，散落在民间的各个角落，它们是一段历史的见证，是我们传统文化中的重要组成部分，是我们现代社会急需要保护的民族的“根”。

如今，这些被遗忘的民间艺术财产，正逐渐的引起各部门、各界人士的重视，因此，一项浩大的保护中国传统文化的工程在全国各地展开，而民间艺术是其中之一，并收到了一些很好的效果，很多单位和个人都为此付出了巨大的努力。如河南开封的旅游景点《清明上河园》是一个全面展现宋文化的平台，回复和再现很多历史场景，如民间杂技、绝活，民间特色手工艺品等，清明上河园发展可以成为中国传统文化的保护集散基地，有这么一批为中国传统文化的保护、发展做出贡献的华夏子孙，传统文化一定会历久弥新。
