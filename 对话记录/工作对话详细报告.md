# 汴京烟火 × Gemini 工作对话详细报告

版本日期：2026-05-24

## 1. 报告目的

本报告基于 Gemini 与用户的工作对话记录，系统梳理“汴京烟火”信息图系列从目标定义、叙事框架、版式约束、图表模块、视觉变量到单图细化的完整讨论链路，归纳关键决策、反复调整点与最终形成的可执行规范。对话来源见：[对话记录/汴京烟火信息图设计最终方案.md](对话记录/汴京烟火信息图设计最终方案.md) 与 [对话记录/文档统计与结构分析.md](对话记录/文档统计与结构分析.md)。

---

## 2. 目标与受众共识

### 2.1 项目核心目标

- 以《东京梦华录》为文本母体，把古籍中的市井生活转译为“可看、易懂、有温度”的信息图。
- 强调“汴京烟火”的气质，而非现代对标或宏大叙事。
- 系列图要统一风格与结构，但每张图只讲一件事。

### 2.2 目标受众（用户明确）

- 核心受众：18-35 岁喜爱宋文化、国潮、市井生活美学的年轻人。
- 次要受众：文旅宣传、中小学历史/美育教育、博物馆宣教从业者。
- 专业受众：视觉传达设计从业者、民俗文化研究者、毕设评审老师。

## 3. 系列结构演进与纠偏过程

### 3.1 早期方案及被否定点

- 初始方案包含“现代对标”（如夜经济、外卖系统）与“分卷叙事矩阵”。
- 用户明确否定“现代对标”，要求纯宋代叙事，且不接受碎片化或抽象叙事。

### 3.2 叙事框架的最终定稿

用户最终给出 6 张图的确定框架，并要求每张图只讲一件事：

| 图号 | 主题 | 场景 | 叙事焦点 |
| --- | --- | --- | --- |
| 图1 | 早市·吃什么 | 早市 | 汴京人早上吃什么 |
| 图2 | 午市·买什么 | 相国寺殿庭市集 | 汴京人买什么 |
| 图3 | 日间·穿什么 | 御廊+寺东门大街服饰市集 | 汴京人日常穿什么、买什么服饰 |
| 图4 | 午后·玩什么 | 瓦舍勾栏 | 娱乐活动大全 |
| 图5 | 夜市·吃什么 | 夜市 | 汴京人在夜市吃什么 |
| 图6 | 岁时·节庆 | 年度节庆 | 汴京人全年怎么过节 |

该框架明确要求：有气质、不干不空；有结构、不散不乱；可直接转画面。

## 4. 三段式固定版式的最终标准

对话最终确立了“固定三段式横构图”作为全系列统一骨架：

### 4.1 上方板块（Header）

必须固定包含的模块（最终规范）：

- 主标题：统一格式，如“汴京烟火·早市食味”，固定在上方最左侧，固定字号与宋体风格。
- 副标题与烟火小志：解释场景背景，文字克制。
- 原文引用区：置于上方模块中，但位置需重新排布；字体缩小，背景淡纹，体现“书”的质感。
- 时空锚点：采用“罗盘 + 物候符号”组合，位置锁死在上方最右侧。
- 汴京方位小地图：极简线稿，仅绘汴河、主干道、城墙轮廓与五大地标。
  - 地标预设：州桥、相国寺、御街、瓦舍街区、龙津桥。
  - 当前图所在区域以淡色填充高亮，并附“本图区位·XXX”。

### 4.2 中间插画板块（Illustration）

- 作为视觉重心，面积需最大化（反复强调“插画太小”）。
- 风格为“界画式线稿 + 淡彩”，人物避免写实，强调剪影与笔触。
- 必须体现生机与烟火气，增加“丰富场景 + 多样人群 + 多样摊铺”。
- 插画不应被蒸汽或空白填满，需要留白呼吸，但避免“平均铺满”。

### 4.3 下方板块（Footer）

- 只保留少量文字描述，以图表与涂鸦为主。
- 允许“理性统计图表 + 叙事性涂鸦”并存。
- 红纸封签固定在底部最右侧，作为系列“符号演变”锚点。

---

## 5. 图表层与涂鸦的统一策略

### 5.1 图表层模块化标准（被要求必须落实）

- 每张图下方配置 2-3 个固定模板，不可过多堆叠。
- 图表可以位于下方，也可在上方模块辅助呈现。
- 禁止气泡图。

### 5.2 理性统计与模糊表达共存

- 统计图表保留具体数值，不能完全放弃量化。
- 对于书中不明确或模糊的数据，允许使用“叙事涂鸦”表达。
  - 例：人群“密集”可用重复小人涂鸦表达。
  - 例：工艺流程可用局部小流程插画表达。

### 5.3 叙事性涂鸦的角色定位

- 不仅是清单，也可作为局部插画或流程示意。
- 涂鸦必须与“场景叙事”一致，具象可读。

## 6. 全周期锚点（符号演变）

用户明确提出“红纸封签”作为全系列符号演变锚点，要求：

- 位置：固定在底部板块最右侧。
- 形态：宋代语境下的纸签/价签/货签，不得现代化。
- 每张图的封签内容与身份不同，但保持风格一致。

示例演变逻辑（对话中形成共识）：

| 图号 | 封签身份 | 语境定位 |
| --- | --- | --- |
| 图1 | 早市价目单 | 早点铺结算单 |
| 图2 | 市集货签 | 相国寺商品清单 |
| 图3 | 服饰料签 | 服装材质/款式标记 |
| 图4 | 入场凭证 | 瓦舍票签或演出名录 |
| 图5 | 夜市包装签 | 夜食封签 |
| 图6 | 祈福签 | 节庆愿签 |

## 7. 图1（早市）细化成果与改动轨迹

### 7.1 早市物料与店铺要求

用户补充并强调必须出现：

- 王楼：山洞梅花包子
- 鹿家包子：各色肉菜包子
- 曹婆婆：煎肉饼
- 沿河沿街：瓠羹老店、灌肺炒肺专业摊
- 液态早餐：分茶点心、煎点汤茶药

### 7.2 图1下方板块数据要求

- 统计图表 A：早市业态占比（晨食摊铺 65% / 茶汤分茶 20% / 杂物小贸 15%）。
- 统计图表 B：早市美食种类占比（具体构成随数据对齐）。
- 叙事性涂鸦：三五文钱购买力；猪/羊食材分解或对应的局部流程。
- 液态早餐流程：分茶点心、煎点汤茶药要做“局部小流程”插画。

### 7.3 插画布局的关键修正点

- 插画必须更大、更丰富、更生机勃勃，不允许“平均铺满”。
- 视角需有主次（主锚点 + 次锚点 + 留白呼吸）。
- 删除“骑马进城”的元素，聚焦“吃什么”的故事。
- 蒸笼蒸汽需减弱，避免遮蔽细节。

## 8. 关键冲突与纠偏记录（摘要）

| 冲突点 | 用户要求 | 调整结果 |
| --- | --- | --- |
| 现代对标 | 明确拒绝 | 回归纯宋代叙事 |
| 叙事方式 | 一图一事 | 六图明确主题 |
| 视觉风格 | 不抽象，宋代气质 | 界画线稿 + 淡彩 |
| 版式结构 | 固定三段式 | 上/中/下模块化锁定 |
| 图表形式 | 多样但统一 | 2-3固定模板，禁气泡图 |
| 图文比例 | 插画过小 | 插画面积扩大，板块压缩 |
| 字体规范 | 必须宋体 | 标题统一“汴京烟火·X” |
| 锚点位置 | 时空锚点位置锁死 | 上方最右侧 |
| 地图准确性 | 需真实区位 | 极简地图 + 高亮区位 |
