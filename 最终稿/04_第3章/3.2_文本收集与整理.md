## 3.2 文本收集与整理

本课题的文本收集以《东京梦华录》原文为主要对象。前期调研中，主要阅读了孟元老《东京梦华录》、邓之诚《东京梦华录注》、伊永文《东京梦华录笺注》、周宝珠《宋代东京研究》，并参考了《〈东京梦华录〉饮食词汇考释》《宋代节日生活研究》《北宋东京休闲娱乐活动研究》《〈清明上河图〉和〈东京梦华录〉中的北宋汴京城市经济》等相关研究成果。这些资料分别涉及原文校注、城市空间、饮食词汇、节令民俗、娱乐活动和商业经济等内容，为后续资料筛选提供了文本范围。

资料收集并非对全书内容进行无差别摘录，而是围绕“汴京烟火”的设计主题，优先选取能够转化为图表、地图、图标和场景插画的信息。收集对象主要包括地点名称、时间表达、人物身份、器物名称、活动内容、饮食种类、服饰形制和节庆场景等内容。对于能够直接支撑图面表达的资料，摘录时同步记录出处、关键词和初步类别，便于后续进入表格整理。这样的处理方式能够避免资料堆积，也能使文本内容更快转化为可用于设计分析的材料。

资料整理采用人工阅读与数字工具辅助相结合的方式。NotebookLM 用于对上传文献进行主题线索梳理，帮助定位与设计主题相关的内容；VS Code 用于电子文本的关键词检索和段落查找，提高摘录效率。这些工具只承担辅助检索和归纳作用，不能替代资料判断。凡是进入数据表的内容，仍需回到原文、注释本或相关研究资料中核对，以保证资料来源可以追溯。

经过前期整理，文本资料被归纳为空间、时辰、服饰、饮食和节庆五类基础方向。空间类资料主要用于整理桥梁、河道、街巷和城市节点；时辰类资料用于提取一日活动的时间线索；服饰、饮食和节庆资料则分别对应衣冠制度、城市消费和岁时民俗等内容。上述五类资料共同构成基础数据集的来源，为后续字段统一和专项数据集制作提供了基础。
