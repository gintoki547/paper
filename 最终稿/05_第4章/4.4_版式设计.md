## 4.4 版式设计

本课题的六张系列信息图均采用横版构图，成品尺寸为 500mm×900mm。版式设计以“统一结构、分图变化”为基本原则，使系列作品在整体视觉上保持一致，同时又能根据不同主题调整信息重点。由于每张信息图都包含标题、场景插画、地图、图表、注释和辅助符号等内容，如果缺少统一的版面规则，容易造成画面信息分散。因此，在设计前期需要先确定基本版式，再根据各分图内容进行局部调整。

整体版面采用分层处理方式。画面上方主要放置系列标题、分图名称和时间提示，使观众能够快速判断作品主题；中部为主要视觉区域，以场景插画、地图或核心图形作为主要阅读对象；下方则安排统计图表、文字说明和资料注释，用于补充理性信息。这样的结构能够兼顾视觉吸引力和信息阅读顺序，使观众先进入主题场景，再进一步阅读数据和说明内容。

在具体排版中，各类信息元素需要形成明确的主次关系。标题区承担主题识别作用，字号和位置相对稳定；主体插画区承担叙事和氛围营造作用，面积相对较大；图表区则强调数据和分类关系，尽量保持清晰、简洁；注释文字以辅助说明为主，避免干扰主体画面。通过标题、场景、图表和注释之间的层级划分，画面既能保持传统文化题材的视觉气质，也能符合信息图形设计对阅读效率的要求。

为保证六张作品在尺寸、色彩、图层和信息区域上的统一性，设计过程中建立了图面规范文档。该文档以 JSON 结构记录画布尺寸、标题区位置、主视觉区域、图表区域、色彩设定和辅助元素等内容，用于约束后续画面制作。正文中仅选取局部结构作为示例，完整规范文件放入附录。

**JSON 规范片段示例：版式结构**

```json
{
  "canvas": {
    "size": "500mm × 900mm",
    "orientation": "horizontal",
    "background": "aged_paper_tone"
  },
  "layout": {
    "title_area": "top",
    "main_visual_area": "middle",
    "chart_area": "bottom",
    "annotation_area": "bottom_side"
  },
  "visual_rules": {
    "title_style": "handwritten_calligraphy",
    "body_font": "Source Han Serif",
    "color_basis": "Song mineral pigment palette",
    "line_style": "fine ink line"
  }
}
```

该规范主要用于控制六张信息图的基础版式，保证标题、插画、图表和注释区域在系列作品中保持一致。具体制作时，再根据不同主题的信息量和画面重点进行局部调整。
